IAIMPUIComboBox
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIBaseEdit -> IAIMPUIBaseButtonnedEdit ->IAIMPUIBaseComboBox -> IAIMPUIComboBox.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_COMBOBOX_PROPID_AUTOCOMPLETE
Когда пользователь вводит текст, выпадающий список автоматически позиционируется на наиболее подходящем элементе. 0 - выключено.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_COMBOBOX_PROPID_ITEMINDEX
Индекс выбранного элемента.
Если ничего не выбрано или введено пользовательское значение - индекс будет равен -1.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_COMBOBOX_PROPID_ITEMOBJECT
Выбранный объект, если ничего не выбрано или введено пользовательское значение - функция чтения свойства вернет ошибку.
ТипIUnknown
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IUnknown, ..)
AIMPUI_COMBOBOX_PROPID_TEXT
Текстовое значение
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_COMBOBOX_PROPID_STYLE
Стиль редактора:
0 - с поддержкой ввода пользовательских значений
1 - без поддержки ввода пользовательских значений
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

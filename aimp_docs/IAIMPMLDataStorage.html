IAIMPMLDataStorage
Описание
Интерфейс предоставляет доступ к свойствам, общим для всех хранилищ данных. На этом же уровне реализуется IAIMPMLGroupingPresets.
Предок
IAIMPMLDataStorage наследуется от IAIMPPropertyList
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPML_DATASTORAGE_PROPID_ID
Уникальный идентификатор хранилища данных
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPString, ..)
AIMPML_DATASTORAGE_PROPID_CAPTION
Название хранилища данных. Отображается пользователю.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPString, ..)
AIMPML_DATASTORAGE_PROPID_GROUPINGPRESET
Активный шаблон дерева группировки
ТипIAIMPMLGroupingPreset
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPMLGroupingPreset, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(.., <IAIMPMLGroupingPreset>)
Создание объекта
<IAIMPServiceMusicLibrary>.GetStorageByID(.., IID_IAIMPMLDataStorage, ..)

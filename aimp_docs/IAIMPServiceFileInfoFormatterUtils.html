IAIMPServiceFileInfoFormatterUtils
Описание
Сервис дает возможность показать контекстное меню со списком доступных макросов (с возможностью выбора пользователем).
Методы
ShowMacrosLegend
inScreenTargetTRect Положение элемента управления в координатах рабочего стола, для которого показывается меню.
При этом контекстное меню не при каких условиях не перекроет указанный прямоугольник.

В случае, если меню нужно показать строго под курсором мыши, без учета положения элемента управления - необходимо указать:

ScreenTarget.Left = ScreenTarget.Right = Cursor.X;
ScreenTarget.Top = ScreenTarget.Bottom = Cursor.Y;
inFormatterIDIntegerОдин из следующих ID:
AIMP_FILEINFO_FORMATTER_ID_BASIC (по умолчанию)
AIMP_FILEINFO_FORMATTER_ID_PLAYING

Форматировщик для проигрываемого файла реализует несколько дополнительных макросов, которые доступны только для проигрываемого файла. Используйте IAIMPServicePlayer.GetInfo, чтобы получить информацию о проигрываемом файле.
inEventsHandlerIUnknown Обработчик выбора макроса - IAIMPActionEvent.
В качестве параметра Data в обработчик приходит IAIMPString с именем макроса.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceFileInfoFormatterUtils, ...);

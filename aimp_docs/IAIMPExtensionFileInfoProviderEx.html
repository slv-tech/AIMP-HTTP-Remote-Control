IAIMPExtensionFileInfoProviderEx
Описание
IAIMPExtensionFileInfoProviderEx представляет собой расширение для сервиса IAIMPServiceFileInfo, предоставляющее расширенную информацию об указанном файле.

Этот интерфейс представляет более общий вид интерфейса IAIMPExtensionFileInfoProvider, в отличие от оригинала, позволяет получить информацию из IAIMPStream, но работает более медленно.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
GetFileInfo
Метод возвращает расширенную информацию о потоке
inStreamIAIMPStream
inInfoIAIMPFileInfo
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

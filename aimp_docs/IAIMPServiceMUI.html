IAIMPServiceMUI
Описание
IAIMPServiceMUI сервис предоставляет доступ к активной на данный момент локализации.
Методы
GetName
Метод возвращает имя локализации - имя файла локализации без пути и расширения.
Для получения "friendly name" воспользуйтесь методом GetValue c ключом "File\Name";
outValueIAIMPString
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetValue
inKeyPathIAIMPStringПуть к переменной в формате <Секция>\<Имя переменной>
outValueIAIMPString
outResultHRESULTS_OK - Метод успешно завершился
E_ACCESSDENIED - Метод был вызван не из главного потока
E_FAIL - Переменная по указанному пути не найдена
E_INVALIDARG - Неправильно задан путь к переменной
E_UNEXPECTED - Произошел непредвиденный сбой
GetValuePart
Некоторые переменные могут содержать в себе несколько значений.
Например: 
        Form.DockToEdges=к левому краю экрана|к правому краю экрана|к верхнему краю экрана|к нижнему краю экрана
Этот метод позволяет получить доступ к каждому из значений по его порядковому номеру.
inKeyPathIAIMPStringПуть к переменной в формате <Секция>\<Имя переменной>
inPartInteger Номер значения, начинается с 0.
outValueIAIMPString
outResultHRESULTS_OK - Метод успешно завершился
E_ACCESSDENIED - Метод был вызван не из главного потока
E_FAIL - Переменная по указанному пути не найдена
E_INVALIDARG - Неправильно задан путь к переменной
E_UNEXPECTED - Произошел непредвиденный сбой
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceMUI, ...);

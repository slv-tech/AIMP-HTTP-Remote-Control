IAIMPMLGroupingTreeDataProvider
Описание
Интерфейс предоставляет возможность плагину реализовать собственный провайдер данных для дерева группировки.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
AppendFilter
Плагин должен построить фильтр для данных для таблицы фонотеки на основе текущей выборки Selection.
inFilterIAIMPMLDataFilterGroup
inSelectionIAIMPMLGroupingTreeSelection
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
GetCapabilities
outResultLongWord 0 или комбинация флагов:
  • AIMPML_GROUPINGTREEDATAPROVIDER_CAP_HIDEALLDATA - не показывать пункт "(показать все)" в дереве группировки
  • AIMPML_GROUPINGTREEDATAPROVIDER_CAP_DONTSORT - выключает автоматическую сортировку записей в дереве группировки
    • GetData
    Возвращает данные для следующего уровня дерева группировки
    inSelectionIAIMPMLGroupingTreeSelection
    outDataIAIMPMLGroupingTreeDataProviderSelection
    outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
    GetFieldForAlphabeticIndex
    inFieldNameIAIMPString Возвращает имя поля верхнего уровня для построения алфавитного уровня.
    Если в вашем случае, верхний уровень состоит из разных полей - алфавитный указатель не сможет работать корректно, поэтому верните E_FAIL в качестве результата.
    outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.

    В случае ошибки считается, что алфавитный указатель недоступен

IAIMPExtensionAlbumArtProvider3
Описание
IAIMPExtensionAlbumArtProvider3 представляет собой расширение для сервиса IAIMPServiceAlbumArt, и позволяет плагину добавить в программу свой провайдер обложек альбома.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
Get
inFileInfoIAIMPFileInfo
inRequestIAIMPAlbumArtRequestПользовательские настройки поиска обложек альбомов, см. IAIMPAlbumArtRequest.

Внимание! Перед выполнение запроса провайдер обязан проверить текущие настройки, и, в случае, если они не удовлетворяют его потребностям, завершить работу с ошибкой E_FAIL.

Рассмотрим взаимодействие с IAIMPAlbumArtRequest на примере провайдера, скачивающего картинку из интернета:

function Get(FileInfo: IAIMPFileInfo; Request: IAIMPAlbumArtRequest; out Image: IAIMPImageContainer): HRESULT; stdcall;
var
  ALink: IAIMPString;
begin
  Result := E_FAIL;
  // Проверяем, задана ли у файла ссылка?
  if PropListGetStr(FileInfo, AIMP_FILEINFO_PROPID_URL, ALink) then
  begin 
    // Это ссылка на картинку?
    if IsImageURL(ALink) then 
    begin
      // Она уже есть в кэше? 
      if Succeeded(Request.CacheGet(ALink, Image)) then
        Exit(S_OK);
        
      // Скачиваем картинку
      // Функция Download самостоятельно проверит и учтёт значения опций 
      // AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET и AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET_MAX_FILE_SIZE
      if Succeeded(Request.Download(ALink, Image)) then
      begin
        // Оптимизируем и обновляем кэш 
        Request.CachePut(ALink, Image);
        Result := S_OK;
      end;
    end;
  end;
end;
outImageIAIMPImageContainer
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
GetCategory
Метод позволяет движку программы узнать, к какой категории относится провайдер.
outResultLongWord Одно из значений:

  • AIMP_ALBUMART_PROVIDER_CATEGORY_TAGS - обложка извлекается из метаданных файла


  • AIMP_ALBUMART_PROVIDER_CATEGORY_FILE - обложка предоставляется в виде внешнего файла, лежащего где-либо на диске (например, рядом с аудиофайлом)


  • AIMP_ALBUMART_PROVIDER_CATEGORY_INTERNET - обложка скачивается с интернета


  • AIMP_ALBUMART_PROVIDER_CATEGORY_INTERNET_SEARCH - обложка скачивается с интернета, сигнализирует о поддержке свойства AIMP_ALBUMART_REQUEST_PROPID_RESULTS (доступно с v5.40)
    • Создание экземпляра
    Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

IAIMPMemoryStream
Описание
IAIMPMemoryStream представляет собой вариант IAIMPStream, где все данные хранятся в памяти.
Предок
IAIMPMemoryStream наследуется от IAIMPStream
Методы
GetData
outResultPByte Указатель на первый элемент массива байт.
Создание объекта
<IAIMPCore>.CreateObject(IID_IAIMPMemoryStream, ...)

Резюме

Объявление:apiObjects.pas / apiObjects.h

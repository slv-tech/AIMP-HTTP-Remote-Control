Внимание
Подписка на события доступна только для 32-разрядных приложений



Подписка на события
Подписка / отписка от событий происходят посредством специальных команд AIMP_RA_CMD_REGISTER_NOTIFY и AIMP_RA_CMD_UNREGISTER_NOTIFY.

Программа должна передать в качестве LParam-а дескриптор окна, которому плеер будет посылать сообщение WM_AIMP_NOTIFY. WParam этого сообщения будет содержать в себе идентификатор события (см. раздел "События").



Пример:
    Window = FindWindow(AIMPRemoteAccessClass, null);
    ...
    // Подписка на обработку событий: 
    SendMessage(Window, WM_AIMP_COMMAND, AIMP_RA_CMD_REGISTER_NOTIFY, Form1.Handle);

    // Отписка от обработки событий: 
    SendMessage(Window, WM_AIMP_COMMAND, AIMP_RA_CMD_UNREGISTER_NOTIFY, Form1.Handle);




    // Обработка событий:
    TForm1 = class(TForm)
    protected
      procedure WMAIMPNotify(var Message: TMessage); message WM_AIMP_NOTIFY;
    end;

    procedure TForm1.WMAIMPNotify(var Message: TMessage);
    begin
      case Message.WParam of
        AIMP_RA_NOTIFY_TRACK_START:
          // что-то делаем
        AIMP_RA_NOTIFY_TRACK_INFO:
          // что-то делаем
        AIMP_RA_NOTIFY_PROPERTY:
          case Message.LParam of
            AIMP_RA_PROPERTY_PLAYER_POSITION:
              // что-то делаем
          end;
      end;
    end;

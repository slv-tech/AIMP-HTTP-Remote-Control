IAIMPExtensionEmbeddedVisualization
Описание
Представляет собой расширение для сервиса IAIMPServiceVisualizations - визуализацию, отображаемую на дисплее визуализации в скине плеера и/или полноэкранном режиме.
Методы
GetFlags
outResultInteger Возвращает параметры визуализации в виде комбинации флагов:
  • AIMP_VISUAL_FLAGS_RQD_DATA_WAVEFORM - плагину необходимы WaveForm-данные для отрисовки
  • AIMP_VISUAL_FLAGS_RQD_DATA_SPECTRUM - плагину необходимы Spectrum-данные для отрисовки
  • AIMP_VISUAL_FLAGS_NOT_SUSPEND - плагин должен рендерить визуализацию всегда, независимо от того, виден дисплей визуализации или нет
GetMaxDisplaySize
Возвращает максимальные поддерживаемые расширением высоту и ширину.
Если целевой дисплей больше поддерживаемых размеров - движок плеера будет автоматически масштабировать визуализацию при выводе.
outWidthInteger Максимальная ширина
outHeightInteger Максимальная высота
outResultHRESULT S_OK - в случае успеха
E_FAIL - если расширение не имеет ограничений на размер дисплея
GetName
outSIAIMPString Название визуализации, отображается в интерфейсе программы.
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
Initialize
Инициализация визуализации, вызывается, когда пользователь активирует визуализацию.
inWidthInteger Ширина дисплея. В процессе рендеринга может меняться, плагин должен корректно обработать событие IAIMPExtensionEmbeddedVisualization.Resize
inHeightInteger Высота дисплея. В процессе рендеринга может меняться, плагин должен корректно обработать событие IAIMPExtensionEmbeddedVisualization.Resize
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
Finalize
Финализация визуализации, позволяет плагину освободить ресурсы, что нужны для рендеринга. Сам плагин при этом финализироваться не должен.

Вызывается, когда пользователь переключается на другую визуализацию или вовсе отключает их отображение.
Click
Клик мышью по дисплею визуализации
inXInteger
inYInteger
inButtonInteger
  • AIMP_VISUAL_CLICK_BUTTON_LEFT - клик левой кнопкой мыши
  • AIMP_VISUAL_CLICK_BUTTON_MIDDLE - клик средней кнопкой мыши
  • Правая кнопка мыши зарезервирована дисплеем визуализации для показа контекстного меню
    • Draw

    Размер контекста определяется процедурой Resize.
    inDataPAIMPVisualData Указатель на данные для визуализации (см. TAIMPVisualData).
    Только для чтения.

    inNewWidthIntegerНовая ширина
    inNewHeightIntegerНовая высота
    Создание экземпляра
    Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

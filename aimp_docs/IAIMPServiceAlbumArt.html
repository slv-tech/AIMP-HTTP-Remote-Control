IAIMPServiceAlbumArt
Описание
Сервис позволяет получить обложку альбома для указанного файла и/или альбома.
Поиск обложки выполняется в отдельном потоке, плагин получит уведомление о завершении операции посредством функции обратного вызова.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:
Доступ к IAIMPPropertyList осуществляется через <IAIMPServiceAlbumArt>.QueryInterface(IID_IAIMPPropertyList, ..);

Список масок (без расширения) для поиска изображения среди файлов в папке.
Каждая маска отделяется от другого точкой с запятой.

Каждое расширение отделяется от другого точкой с запятой.

AIMP_ALBUMART_REQUEST_PROPID_RESULTS
Указывает число результатов поиска, которые может вернуть провайдер посредством вызова IAIMPAlbumArtRequest.CachePut в ходе одного запроса.
Только для провайдеров категории AIMP_ALBUMART_PROVIDER_CATEGORY_INTERNET_SEARCH.

inFlagsLongWord 0 или комбинация из флагов:
  • AIMP_SERVICE_ALBUMART_FLAGS_NOCACHE - поиск обложки в обход кэша
  • AIMP_SERVICE_ALBUMART_FLAGS_ORIGINAL - метод вернет обложку альбома в ее оригинальном разрешении
  • AIMP_SERVICE_ALBUMART_FLAGS_WAITFOR - метод не вернет управление до тех пор, пока операция не будет завершена
  • AIMP_SERVICE_ALBUMART_FLAGS_OFFLINE - принудительно отключает поиск обложек в Интернете

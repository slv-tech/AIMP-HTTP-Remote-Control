IAIMPUIProgressDialogEvents
Методы
OnCanceled
Метод вызывается, когда пользователь нажимает на кнопку "Отмена".
Если плагин не реализует этот интерфейс - "отмена" будет недоступна пользователю.
Создание экземпляра
Экземпляр создается плагином и передается в метод <IAIMPServiceUI>.CreateObject при создании диалога.

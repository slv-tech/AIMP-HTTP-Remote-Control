IAIMPServiceFileInfoFormatter
Описание
Сервис позволяет вывести информацию о файле в виде текста согласно указанной форматной строке.
Методы
Format
inTemplateIAIMPString Форматная строка.

Список поддерживаемых макросов форматной строки может зависеть от конкретной версии программы.
inFileInfoIAIMPFileInfo Информация о файле.
Укажите nil/null в качестве параметра, чтобы сгенерировать строку для предпросмотра.
inFormatterIDIntegerОдин из следующих ID:
AIMP_FILEINFO_FORMATTER_ID_BASIC (по умолчанию)
AIMP_FILEINFO_FORMATTER_ID_PLAYING

Форматировщик для проигрываемого файла реализует несколько дополнительных макросов, которые доступны только для проигрываемого файла. Используйте IAIMPServicePlayer.GetInfo, чтобы получить информацию о проигрываемом файле.
inAdditionalInfoIUnknown Дополнительная информация о файле. Для будущего использования, должно быть равно nil / null.
outFormattedResultIAIMPString Результат работы функции.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceFileInfoFormatter, ...);

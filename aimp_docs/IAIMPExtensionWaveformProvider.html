IAIMPExtensionWaveformProvider
Описание
Это расширение сервиса IAIMPServiceWaveform позволяет плагину зарегистрировать собственный провайдер данных для waveform-навигатора

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
Calculate
inFileURIIAIMPStringFileURI
inTaskOwnerIAIMPTaskOwner Интерфейс позволяет узнать об отмене задачи пользователем.
См. рекомендации по реализации.

inPeaksPAIMPWaveformPeakInfoArray Указатель на первый элемент массива TAIMPWaveformPeakInfo.

Плагин должен посчитать "волну" для FileURI с разрешением в PeakCount точек, и записать ее в этот массив.
inPeakCountInteger Количество элементов в массиве Peaks
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

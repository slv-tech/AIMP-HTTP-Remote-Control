IAIMPServiceOptionsDialog
Описание
Сервис позволяет создать собственную вкладку в диалоге настроек программы. Вкладка будет отображаться в группе "Плагины".
Методы
FrameModified
При изменении каких-либо опций пользовательская вкладка должна уведомить диалог настроек с помощью этого метода.
inFrameIAIMPOptionsDialogFrameВкладка, на которой произошли изменения
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
FrameShow
Позволяет программно переключиться на указанную вкладку
inFrameIUnknownIAIMPOptionsDialogFrame
inForceShowLongBool Принудительно запускает диалог настроек, если он не открыт.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceOptionsDialog, ...);

TAIMPUIControlPlacement
Общие сведения
TAIMPUIControlPlacement содержит в себе настройки расположения элемента управления.

Движок плеера предусматривает несколько режимов позиционирования элементов:
1. Абсолютное позиционирование (используемые свойства: Bounds)
2. Позиционирование с якорной привязкой (используемые свойства: Anchors, Bounds)
3. Позиционирование с выравниванием по стороне (используемые свойства: Alignment, AlignmentMargins, Bounds)
Описание
TAIMPUIControlPlacement
varAlignmentTAIMPUIControlAlignment Может содержать одно из значений:
+ ualNone - выравнивание не используется
+ ualTop - выравнивание по верхнему краю
+ ualBottom - выравнивание по нижнему краю
+ ualLeft - выравнивание по левому краю
+ ualRight - выравнивание по правому краю
+ ualClient - элемент занимает все оставшееся пространство
varAlignmentMarginsTRect Отступы от соседей, только 0 и положительные значения. Используется при Alignment отличном от ualNone.
varAnchorsTRect Состояние якоря для каждой из сторон, 0 - якорь не назначен. Используется при Alignment = ualNone.
varBoundsTRect Абсолютная позиция элемента управления.
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

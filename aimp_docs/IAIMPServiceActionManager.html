IAIMPServiceActionManager
Описание
Сервис предоставляет доступ к командам общего назначения.
Методы
GetByID
inIDIAIMPStringИдентификатор команды, см. IAIMPAction
outActionIAIMPActionИнтерфейс команды
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
MakeHotkey
Сервисная функция, помогает создать идентификатор комбинации горячих клавиш.
inModifiersWord Комбинация из следующих флагов:
  • AIMP_ACTION_HOTKEY_MODIFIER_CTRL
  • AIMP_ACTION_HOTKEY_MODIFIER_ALT
  • AIMP_ACTION_HOTKEY_MODIFIER_SHIFT
  • AIMP_ACTION_HOTKEY_MODIFIER_WIN
Внимание! Модификатор AIMP_ACTION_HOTKEY_MODIFIER_WIN поддерживается только для глобальных горячих клавиш.
inKeyWord Виртуальный код клавиши, см. константы VK_XXX из Windows SDK.
outResultInteger Уникальный идентификатор горячей клавиши
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceActionManager, ...);

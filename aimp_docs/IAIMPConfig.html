IAIMPConfig
Описание
Интерфейс представляет базовый функционал для работы с конфигурационными файлами.
Методы
Delete
Функция позволяет удалить значение переменной или секцию целиком
inKeyPathIAIMPString Путь к переменной или секции.

Например:
  • <Секция>
  • <Секция>\<Имя переменной>
outResultHRESULT S_OK - Метод успешно завершился
E_ACCESSDENIED - Метод был вызван не из главного потока
E_FAIL - Переменная или секция по указанному пути не найдены
E_INVALIDARG - Неправильно задан путь к свойству или секции
E_UNEXPECTED - Произошел непредвиденный сбой

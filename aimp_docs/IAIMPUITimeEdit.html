IAIMPUITimeEdit
Описание
Поле для ввода времени.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUITimeEdit
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_TIMEDIT_PROPID_VALUE
Текущее значение.

Целая часть - дни, дробная часть - миллисекунды.
Формат времени совместим с типом TDateTime в Delphi.
ТипDouble
ЧтениеIAIMPPropertyList.GetValueAsFloat
ЗаписьIAIMPPropertyList.SetValueAsFloat
События
IAIMPUIChangeEvents
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

IAIMPUIBaseButtonnedEdit
Описание
IAIMPUIBaseButtonnedEdit представляет собой базовый интерфейс для всех полей ввода с встроенными кнопками.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIBaseEdit -> IAIMPUIBaseButtonnedEdit
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_BUTTONEDEDIT_BUTTONSIMAGES
Набор картинок для встроенных кнопок
ТипIAIMPUIImageList
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPUIImageList, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPUIImageList>)

IAIMPUIImageComboBox
Описание
IAIMPUIImageComboBox - поле с выпадающим списком, к каждой записи которого можно назначить картинку. Не поддерживает ввод пользовательских данных.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIBaseEdit -> IAIMPUIBaseButtonnedEdit ->IAIMPUIBaseComboBox -> IAIMPUIImageComboBox
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_IMAGECOMBOBOX_PROPID_IMAGELIST
Лист с картинками для записей редактора
ТипIAIMPUIImageList
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPUIImageList, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPUIImageList>)
AIMPUI_COMBOBOX_PROPID_ITEMINDEX
Индекс выбранного элемента. Если ничего не выбрано - индекс будет равен -1.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32

Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

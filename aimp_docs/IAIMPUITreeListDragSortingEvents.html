IAIMPUITreeListDragSortingEvents
Описание
События группы IAIMPUITreeListDragSortingEvents возникают при ручной сортировке (если она разрешена) узлов.

Методы
OnDragSorting
Событие возникает после завершения сортировки с помощью drag-n-drop.
inSenderIAIMPUITreeList
OnDragSortingNodeOver
Событие возникает при попытке переместить выбранные узлы в указанную позицию.
inSenderIAIMPUITreeList
inNodeIAIMPUITreeListNode Узел, над которым сейчас расположена мышь
inFlagsLongWord Показывает в какую именно позицию будут вставлены выбранные узлы.

Один из следующих флагов:
AIMPUI_FLAGS_TL_INSERTTO_AFTER
AIMPUI_FLAGS_TL_INSERTTO_BEFORE
AIMPUI_FLAGS_TL_INSERTTO_INTO
varAllowLongBool Выставите переменную в True, если перемещение в указанную позицию допустимо.

IAIMPMLDataStorageCommandFindInLibrary
Описание
Команда помогает плееру определить, где в фонотеке расположен указанный пользователем файл.
Если файл принадлежит хранилищу данных плагина - плагин должен заполнить выходные параметры и вернуть S_OK.
Методы
Find
inFileInfoIAIMPFileInfo Информация о файле, который нужно найти в фонотеке.
outGroupingPresetIDIAIMPStringID шаблона дерева группировки, на который нужно переключиться, чтобы применить данные из параметра GroupingTreeSelection.
Верните nil/null, если изменять текущий шаблон группировки не нужно.
outGroupingTreeSelectionIAIMPMLGroupingTreeSelection Путь к узлу дерева группировки, где находится указанный пользователем файл.
Чтобы выбрать корень дерева, верните в качестве значения nil/null или пустой список.
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.

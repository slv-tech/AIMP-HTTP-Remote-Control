IAIMPExtensionLyricsProvider
Описание
IAIMPExtensionLyricsProvider представляет собой расширение для сервиса IAIMPServiceLyrics, и позволяет плагину добавить в программу свой провайдер для текстов песен.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
Get
inOwnerIAIMPTaskOwner Интерфейс позволяет узнать об отмене задачи пользователем.
См. рекомендации по реализации.

inFileInfoIAIMPFileInfo
inFlagsLongWord Флаги соответствуют флагам, переданным в IAIMPServiceLyrics.Get
inLyricsIAIMPLyrics Контейнер для найденной информации, должен заполняться провайдером только в случае успешного завершения операции поиска.
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
GetCategory
Метод сообщает движку программы категорию провайдера
outResultLongWord Одно из значений:
  • AIMP_LYRICS_PROVIDER_CATEGORY_FILE
  • AIMP_LYRICS_PROVIDER_CATEGORY_INTERNET
    • Создание экземпляра
    Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

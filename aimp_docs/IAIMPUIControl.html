IAIMPUIControl
Описание
IAIMPUIControl является базовым элементов для всех элементов управления.
Предок
IAIMPPropertyList -> IAIMPUIControl.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_CONTROL_PROPID_CUSTOM
Пользовательский параметр, позволяет прикрепить к экземпляру интерфейса пользовательский набор данных
ТипIUnknown
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IUnknown, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IUnknown>)
AIMPUI_CONTROL_PROPID_ENABLED
Доступность элемента: 0 - недоступен, иначе - любое другое значение.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_CONTROL_PROPID_HINT
Текст подсказки, отображается при удержании мыши над элементом.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_CONTROL_PROPID_NAME
Имя элемента, может содержать только латинские буквы и цифры. Используется для автоматической локализации
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_CONTROL_PROPID_PARENT
Контейнер, на котором расположен элемент.
ТипIAIMPUIWinControl
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPUIWinControl, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPUIWinControl>)
AIMPUI_CONTROL_PROPID_POPUPMENU
Контекстное меню элемента, может быть не задано.
ТипIAIMPUIPopupMenu
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPUIPopupMenu, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPUIPopupMenu>)
AIMPUI_CONTROL_PROPID_VISIBLE
Видимость элемента: 0 - невиден, иначе - любое другое значение.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32

Объект поддерживает IAIMPDPIAware. Интерфейс позволяет определить текущей DPI элемента управления.

Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

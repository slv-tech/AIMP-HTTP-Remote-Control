IAIMPUISpinEdit
Описание
Поле для ввода целых или дробных числовых значений, по бокам которого расположены кнопки для уменьшения / увеличения введенного значения.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUISpinEdit
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_SPINEDIT_PROPID_DISPLAYMASK
Форматная строка для отображаемого значения. В строке обязательно должен присутствовать макрос %s, на его место будет вставлено текущее значение редактора.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_SPINEDIT_PROPID_VALUETYPE
Тип редактируемого значения:
0 - Int32;
1 - Double;

Тип свойств ниже зависит от этого значения.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_SPINEDIT_PROPID_INCREMENT
Величина, на которую будет увеличиваться / уменьшаться введенное значение при клике по боковым кнопкам
ТипInteger / Double
ЧтениеIAIMPPropertyList.GetValueAsInt32 / IAIMPPropertyList.GetValueAsFloat
ЗаписьIAIMPPropertyList.SetValueAsInt32 / IAIMPPropertyList.SetValueAsFloat
AIMPUI_SPINEDIT_PROPID_MAXVALUE
ТипInteger / Double
ЧтениеIAIMPPropertyList.GetValueAsInt32 / IAIMPPropertyList.GetValueAsFloat
ЗаписьIAIMPPropertyList.SetValueAsInt32 / IAIMPPropertyList.SetValueAsFloat
AIMPUI_SPINEDIT_PROPID_MINVALUE
ТипInteger / Double
ЧтениеIAIMPPropertyList.GetValueAsInt32 / IAIMPPropertyList.GetValueAsFloat
ЗаписьIAIMPPropertyList.SetValueAsInt32 / IAIMPPropertyList.SetValueAsFloat
AIMPUI_SPINEDIT_PROPID_VALUE
Текущее значение
ТипInteger / Double
ЧтениеIAIMPPropertyList.GetValueAsInt32 / IAIMPPropertyList.GetValueAsFloat
ЗаписьIAIMPPropertyList.SetValueAsInt32 / IAIMPPropertyList.SetValueAsFloat
События
IAIMPUIChangeEvents вызывается при изменении значения
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

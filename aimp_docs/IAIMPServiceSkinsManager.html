IAIMPServiceSkinsManager
Методы
EnumSkins
Метод возвращает список имен файлов (без пути) всех установленных скинов
inListIAIMPObjectList Список, содержащий IAIMPString
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
GetSkinInfo
Получение информацию о скине
inFileNameIAIMPString Если в FileName указано только имя файла, то скин-движок автоматически произведет поиск файла с таким именем среди установленных скинов.
outInfoIAIMPSkinInfo
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Select
inFileNameIAIMPString Аргумент может принимать одно из следующих значений:
  • nil / null - переприменить текущий скин
  • Имя файла установленного (без пути) скина - переключиться на указанный скин
  • пустая строка - применить скин по умолчанию

outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Install
Метод устанавливает скин, но не применяет его.
inFileNameIAIMPString Абсолютное имя файла
inFlagsLongWord Комбинация из следующих флагов:
  • AIMP_SERVICE_SKINSMAN_FLAGS_INSTALL_FOR_ALL_USERS
Внимание! Для установки скина в общую для всех пользователей папку нужны права администратора.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Uninstall
Метод удаляет установленный скин, он не изменяет текущую обложку.

Внимание! Для удаления скина из общей для всех пользователей папки нужны права администратора.
inFileNameIAIMPString Относительное имя файла
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
HSLToRGB / RGBToHSL
Сервисные методы для преобразования цвета между пространствами RGB и HSL
Свойства
Помимо перечисленных методов, сервис имеет свойства:
Доступ к списку свойств:
    <IAIMPServiceSkinsManager>.QueryInterface(IID_IAIMPPropertyList, ...);

AIMP_SERVICE_SKINSMAN_PROPID_SKIN
Имя файла (без пути) установленного скина
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPString, ..)
AIMP_SERVICE_SKINSMAN_PROPID_HUE
Текущий оттенок, от 0 до 255. 0 - оттенок не задан.
Для применения эффекта необходимо вызвать метод Select
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_SERVICE_SKINSMAN_PROPID_HUE_INTENSITY
Насыщенность оттенка, от 0 до 100. 0 - насыщенность не задана
Для применения эффекта необходимо вызвать метод Select
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceSkinsManager, ...);

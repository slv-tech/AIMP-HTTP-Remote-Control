IAIMPUIWinControl
Описание
IAIMPUIWinControl, в отличие от IAIMPUIControl, может работать с клавиатурой и принимать в себя другие элементы управления.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl
Свойства
AIMPUI_WINCONTROL_PROPID_FOCUSED
1 - элемент управления сфокусирован, 0 - в противном случае
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
AIMPUI_WINCONTROL_PROPID_TABORDER
Индекс в списке для навигации между элементами управления по клавише Tab
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
Методы
GetControl
Метод позволяет получить доступ к дочерним элементам управления.
inIndexInteger Порядковый номер элемента
inIIDTGUID Идентификатор класса элемента
outObjIUnknown Элемент
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetControlCount
outResultInteger
GetHandle
Метод возвращает оконный дескриптор элемента управления в терминах Windows. Если дескриптор еще не создан - создает его.
outResultHWND
HasHandle
Метод показывает, создан ли оконный дескриптор элемента управления
outResultLongBool
SetFocus
Метод устанавливает фокус элементу управления
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
События
IAIMPUIKeyboardEvents
IAIMPUIWndProcEvents
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

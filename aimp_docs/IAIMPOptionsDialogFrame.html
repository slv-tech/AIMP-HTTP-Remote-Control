IAIMPOptionsDialogFrame
Описание
Интерфейс описывает пользовательскую вкладку, добавляемую в диалог настроек.
Методы
GetName
outSIAIMPString Заголовок вкладки, отображается в интерфейсе программы. Может зависеть от выбранной локализации.
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
CreateFrame
Создание содержимого вкладки.
inParentWindowHWND Дескриптор окна, на котором должно располагаться содержимое вкладки.
Узнать размер окна можно с помощью функции GetWindowRect
outResultHWND Дескриптор окна созданной вкладки
DestroyFrame
Удаление содержимого вкладки.
Notification
Обработчик уведомлений от диалога настроек
inIDInteger Идентификатор уведомления, может принимать одно из следующих значений:
  • AIMP_SERVICE_OPTIONSDIALOG_NOTIFICATION_LOAD - вкладка должна инициализировать элементы управления значениями из конфигурационного файла
  • AIMP_SERVICE_OPTIONSDIALOG_NOTIFICATION_LOCALIZATION - изменилась локализация, вкладка должна сменить локализацию у своих элементов управления
  • AIMP_SERVICE_OPTIONSDIALOG_NOTIFICATION_SAVE - вкладка должна сохранить изменения в конфигурационный файл
  • AIMP_SERVICE_OPTIONSDIALOG_NOTIFICATION_CAN_SAVE - нотификация вызывается перед сохранением с целью проверки выставленных параметров. Если один из параметров выставлен неверно - плагин должен сгенерировать любое исключение. Текст исключения не отображается пользователю.
  • AIMP_SERVICE_OPTIONSDIALOG_NOTIFICATION_RESET - вкладка должна сбросить настройки в значения по умолчанию
DPI Aware
При изменении масштаба окна настроек, дескриптору каждой вкладки (см. CreateFrame) посылается сообщение WM_DPICHANGED. Плагин должен корректно обработать это сообщение, увеличив масштаб элементов на вкладке.
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

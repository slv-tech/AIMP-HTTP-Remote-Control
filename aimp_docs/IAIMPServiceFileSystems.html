IAIMPServiceFileSystems
Описание
Сервис IAIMPServiceFileSystems предоставляет доступ к командам работы с файлами для конкретного файла.
Принцип работы
"Файловая система" - это набор интерфейсов, для работы с файлами с определенным видом пути:
схема://путь или схема:\\путь
Каждое расширение сервиса крепится за конкретной схемой. При какой-либо операции движок программы берет схему файла и ищет интерфейс файловой системы, который ее реализует. Если интерфейс файловой системы не найден - используется интерфейс файловой системы по умолчанию. Затем запрашивает у него интерфейс команды по ее IID (Список доступных команд перечислен в подразделах текущего раздела).
Методы
Get
Метод позволяет получить интерфейс команды по ее IID для файла с конкретной схемой.
inFileURIIAIMPString Имя виртуального файла
inIIDTGUID IID команды
outCommand
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
GetDefault
Метод позволяет получить интерфейс команды по ее IID для схемы по умолчанию (без схемы).
inIIDTGUID IID команды
outCommand
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceFileSystems, ...);

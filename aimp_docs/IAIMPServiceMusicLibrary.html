IAIMPServiceMusicLibrary
Описание
IAIMPServiceMusicLibrary сервис предоставляет общий доступ к хранилищам данных фонотеки.
Методы
GetActiveStorage
Метод возвращает интерфейс по IID для активного хранилища данных
inIIDTGUIDIID запрашиваемого объекта.
Используйте IID_IAIMPMLDataStorage для доступа к обобщенным свойствам источников.
Используйте IID_IAIMPMLGroupingPresets для доступа к списку шаблонов дерева группировки.

GetStorageByID
Метод возвращает экземпляр интерфейса хранилища данных по его ID.
inIDIAIMPString ID хранилища.

См. AIMPML_DATASTORAGE_PROPID_ID для подробностей.

Для доступа к хранилищу локальных файлов, используйте константу AIMPML_LOCALDATASTORAGE_ID

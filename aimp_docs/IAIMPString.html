IAIMPString
Описание
IAIMPString представляет собой высокоуровневую обертку над массивом юникодных-символов.

Этот объект поддерживает IAIMPHashCode.
Методы

inIndexIntegerПорядковый номер запрашиваемого символа (от 0 до GetLength - 1)

E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
GetData
Метод возвращает массив символов, заканчивающийся нулевым символом. Длина массива равна GetLength + 1 (с учетом нулевого).
Нулевой символ позволяет использовать возвращаемый этим методом результат в функциях Win32 API без указания длины.

Внимание! Массив возвращаются "только для чтения"! В случае модификации данных в массиве, работоспособность других методов интерфейса может быть нарушена.

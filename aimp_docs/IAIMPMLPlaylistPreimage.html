IAIMPMLPlaylistPreimage
Описание
Интерфейс предоставляет доступ к настройкам прообраза, что используют одном из хранилищ данных фонотеки в качестве источника.

Обратите внимание, что не все хранилища данных поддерживают создание прообразов.
Предок
IAIMPMLPlaylistPreimage наследуется от IAIMPPlaylistPreimage
Методы
GetFilter
Возвращает ссылку на используемый фильтр.
outFilterIAIMPMLDataFieldFilter Изменения, сделанные в этом интерфейсе, автоматически применяются к прообразу.
Для множественных изменений в фильтре используйте BeginUpdate / EndUpdate.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetStorage
Возвращает интерфейс хранилища данных, к которому привязан прообраз.

IAIMPErrorInfo
Описание
Интерфейс позволяет передать вызывающей стороне получить расширенную информацию об ошибке.
Методы
GetInfo
outErrorCodeIntegerКод ошибки
outMessageIAIMPString Краткое текстовое описание ошибки.
outDetailsIAIMPString Расширенное текстовое описание ошибки. Может быть не задано - nil / NULL
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
GetInfoFormatted
Метод возвращает информацию об ошибке в пригодном для отображения пользователю виде.
outSIAIMPString
outResultHRESULTS_OK - Метод успешно завершился

E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой

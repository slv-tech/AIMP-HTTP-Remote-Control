IAIMPFileSystemCommandStreaming
Описание
Команда создает экземпляр IAIMPStream для указанного FileName.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
CreateStream
inFileNameIAIMPString
inFlagsLongWord Комбинация из следующих флагов:
  • AIMP_SERVICE_FILESTREAMING_FLAG_CREATENEW - перезапись или создание файла.
  • AIMP_SERVICE_FILESTREAMING_FLAG_READ - открытие существующего файла для чтения.
  • AIMP_SERVICE_FILESTREAMING_FLAG_READWRITE - открытие существующего файла для чтения / записи.
  • AIMP_SERVICE_FILESTREAMING_FLAG_BUFFERED - использовать пребуферизацию операций чтения / записи для лучшей производительности
inOffsetInt64 Параметры Offset и Size позволяют создать поток для участка файла.
Если ограничение не требуется - оба параметра должны быть равны -1.

Несовместимы с флагами AIMP_SERVICE_FILESTREAMING_FLAG_CREATENEW и AIMP_SERVICE_FILESTREAMING_FLAG_READWRITE.
inSizeInt64 см. Offset.
outStreamIAIMPStream
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой

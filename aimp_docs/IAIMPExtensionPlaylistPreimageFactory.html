IAIMPExtensionPlaylistPreimageFactory
Описание
Интерфейс позволяет плагину зарегистрировать в программе свою фабрику прообразов (источников данных) для смарт-плейлистов.
Методы
CreatePreimage
Метод создает новый экземпляр прообраза.
Смотрите более подробную информацию о принципах работы плеера с прообразами в описании интерфейса IAIMPPlaylistPreimage.
outIntfIAIMPPlaylistPreimage
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetID
Уникальный ID фабрики. ID не должен меться в процессе развития плагина - этот ID сохраняется в каждый файл смарт-плейлиста.

E_ACCESSDENIED - Метод был вызван не из главного потока

IAIMPUIPopupMenu
Описание
IAIMPUIPopupMenu - невизуальный элемент, позволяющий создавать контекстное меню.
Предок
IAIMPPropertyList -> IAIMPUIPopupMenu
Методы
Add
inIDIAIMPString Уникальный идентификатор меню, используется для автоматической локализации.

outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

События
IAIMPUIPopupMenuEvents вызывается перед показом меню
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateObject.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

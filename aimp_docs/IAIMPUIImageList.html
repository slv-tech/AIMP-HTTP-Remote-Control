IAIMPUIImageList
Описание
IAIMPUIImageList является хранилищем для небольших картинок (до 256х256) для элементов управления.

Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateObject.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).
DPI Aware
Объект поддерживает IAIMPDPIAware. Интерфейс позволяет определить исходное разрешение картинок в листе.

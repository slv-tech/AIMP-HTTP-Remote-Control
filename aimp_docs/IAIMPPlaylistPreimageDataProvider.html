IAIMPPlaylistPreimageDataProvider
Описание
Интерфейс является основополагающим для прообраза - он позволяет получить данные для плейлиста. Должен быть реализован на том же уровне, что и IAIMPPlaylistPreimage.
Методы
GetFiles
inOwnerIAIMPTaskOwner Интерфейс позволяет узнать об отмене задачи пользователем.
См. рекомендации по реализации.

outFlagsLongWord 0 или комбинация из флагов:
  • AIMP_PLAYLIST_ADD_FLAGS_NOCHECKFORMAT - флаг подавляет проверку на поддерживаемые типы при добавлении файлов в плейлист
  • AIMP_PLAYLIST_ADD_FLAGS_NOEXPAND - флаг позволяет добавить в плейлист файлы из списка как есть, не раскрывая их на виртуальные файлы
  • AIMP_PLAYLIST_ADD_FLAGS_NOASYNC - добавление файлов в плейлист будет происходить в главном потоке, управление не будет передано плагину до тех пор, пока операция не будет завершена (не рекомендуется)
  • AIMP_PLAYLIST_ADD_FLAGS_FILEINFO - флаг показывает, что в качестве объекта передан IAIMPFileInfo (в противном случае объект трактуется как IAIMPString)
    • outListIAIMPObjectList Список FileURI в виде IAIMPString или IAIMPFileInfo
    outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.

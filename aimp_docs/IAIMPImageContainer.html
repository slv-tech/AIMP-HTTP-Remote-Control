IAIMPImageContainer
Описание
IAIMPImageContainer в отличие от IAIMPImage служит для хранения и шаринга изображения. Изображение хранится в сжатом виде для экономии памяти, а все параметры вычисляются "на лету".

Этот объект поддерживает IAIMPHashCode.
Методы
CreateImage
Метод создает IAIMPImage, основанный на копии исходных данных.
outImageIAIMPImage
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
GetInfo
Информация об изображении.
Информация определяется по заголовкам формата изображения, хранимых в сжатом потоке, т.е. для получения информации изображение не распаковывается, тем самым экономя вычислительные ресурсы. Однако возвращаемая информация может не соответствовать реальности, если заголовки повреждены.
outSizeTSize Размеры изображения
outFormatIDInteger Идентификатор формата изображения, может принимать одно из значений:
  • AIMP_IMAGE_FORMAT_UNKNOWN
  • AIMP_IMAGE_FORMAT_BMP
  • AIMP_IMAGE_FORMAT_GIF
  • AIMP_IMAGE_FORMAT_JPG
  • AIMP_IMAGE_FORMAT_PNG

IAIMPEqualizerBands
Описание
IAIMPEqualizerBands представляет собой обобщенный интерфейс для доступа к настройке эквалайзера.
Методы
GetBandCount
outResultInteger Количество полос эквалайзера
GetBandFrequency
intBandIndexInteger Порядковый номер полосы, начинается с 0.
outValuedouble Частота полосы в Гц.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetBandGain
intBandIndexInteger Порядковый номер полосы, начинается с 0.
outValuedouble Текущее значение усиления, от -15.0 до +15.0 дБ.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
SetBandGain
intBandIndexInteger Порядковый номер полосы, начинается с 0.
inValuedouble Текущее значение усиления, от -15.0 до +15.0 дБ.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

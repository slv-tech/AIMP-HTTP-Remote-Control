IAIMPActionEvent
Описание
Интерфейс представляет собой обработчик действия. Позволяет плагину получить управление при вызове команды.
Методы
OnExecute
Обработчик действия
inDataIUnknown Дополнительные данные, определяются вызывающей стороной. Параметр может быть равен nil / null.
Создание экземпляра
Экземпляр создается плагином и назначается в IAIMPAction c помощью свойства AIMP_ACTION_PROPID_EVENT.

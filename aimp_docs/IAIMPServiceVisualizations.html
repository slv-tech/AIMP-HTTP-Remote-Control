IAIMPServiceVisualizations
Описание
IAIMPServiceVisualizations не имеет методов, интерфейс введен для возможности регистрации пользовательских расширений.

Пример использования:
<IAIMPCore>.RegisterExtension(IID_IAIMPServiceVisualizations, TMyVisualExtension.Create);
Резюме
Объявление:apiVisuals.pas / apiVisuals.h
Асинхронная работа:Не поддерживается
Расширения:IAIMPExtensionCustomVisualization, IAIMPExtensionEmbeddedVisualization
Приложения:Player
Минимальная версия:3.60.1455

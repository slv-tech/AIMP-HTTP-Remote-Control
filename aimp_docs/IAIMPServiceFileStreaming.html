IAIMPServiceFileStreaming
Описание
Сервис IAIMPServiceFileStreaming представляет собой сервис для доступа к реальным и виртуальным файлам посредством IAIMPStream.
Методы
CreateStreamForFile
Метод создает экземпляр IAIMPStream для указанного реального файла.
Параметры и их описание см. в IAIMPFileSystemCommandStreaming.
CreateStreamForFileURI
Метод создает экземпляр IAIMPStream для указанного виртуального файла.
inFileURIIAIMPString Имя виртуального файла
outVirtualFileIAIMPVirtualFile Информация о виртуальном файле. Может быть равна nil (null).
outStreamIAIMPStream
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceFileStreaming, ...);

Свойство показывает, может ли пользователь менять состояние флажка.
Отображение флажков должно быть включено.

По умолчанию: 1

AIMPUI_TL_NODE_PROPID_CHILDREN_CHECK_STATE
Состояние флажка для дочерних элементов, рассчитывается автоматически.

Может принимать одно из следующих значений:
  • AIMPUI_CHECKSTATE_UNCHECKED
  • AIMPUI_CHECKSTATE_CHECKED
  • AIMPUI_CHECKSTATE_MIXED

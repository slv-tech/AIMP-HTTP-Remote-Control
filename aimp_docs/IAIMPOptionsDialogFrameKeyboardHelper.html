IAIMPOptionsDialogFrameKeyboardHelper
Описание
Интерфейс обеспечивает взаимодействие между фреймом плагина и диалогом, в котором он показывается, при навигации с клавиатуры между элементами интерфейса.

Интерфейс должен реализовываться на том же уровне, что и IAIMPOptionsDialogFrame.
Методы
DialogKey
inCharCodeWord Код символа / клавиши
inUnusedInteger Для будущего использования
outResultLongBool True - если плагин обработал эту комбинацию.
SelectFirstControl
Метод вызывается, когда пользователь заходит на фрейм с клавиатуры с другого элемента управления, что находится вне фрейма плагина.
outResultLongBool Если плагин не нашел элемент управления для фокусировки - метод должен вернуть False, иначе - True.
SelectNextControl
Метод вызывается, когда пользователь с клавиатуры переходит с одного элемента управления на другой в пределах фрейма плагина.
inFindForwardLongBool Направление перемещения фокуса - вперед / назад
inIsTabKeyActionLongBool True - навигация осуществляется нажатием клавиши Tab, иначе - стрелками.
outResultLongBool Если плагин не нашел элемент управления для фокусировки - метод должен вернуть False, иначе - True.

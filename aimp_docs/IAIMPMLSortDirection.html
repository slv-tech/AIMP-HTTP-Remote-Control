IAIMPMLSortDirection
Description
Интерфейс предоставляет доступ к информации о направлении сортировки.
Методы
GetValue
outValueIntegerМожет принимать одно из значений:
  • AIMPML_SORTDIRECTION_ASCENDING
  • AIMPML_SORTDIRECTION_DESCENDING
    • outResultHRESULTS_OK - Метод успешно завершился
    E_INVALIDARG - Один из параметров задан неверно
    E_NOTIMPL - Метод не реализован
    E_UNEXPECTED - Произошел непредвиденный сбой

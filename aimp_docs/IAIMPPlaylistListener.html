IAIMPPlaylistListener
Описание
Интерфейс позволяет плагину получать уведомления об изменениях в плейлисте
Методы
Activated
Событие возникает при активации плейлиста
Changed
inFlagsLongWord Комбинация из флагов:
  • AIMP_PLAYLIST_NOTIFY_NAME - изменилось имя плейлиста
  • AIMP_PLAYLIST_NOTIFY_GROUPNAME - плейлист был перемещен в другую группу (v5.40)
  • AIMP_PLAYLIST_NOTIFY_SELECTION - изменился список выбранных записей
  • AIMP_PLAYLIST_NOTIFY_MODIFIED - плейлист изменился, необходимо пересохранить файл
  • AIMP_PLAYLIST_NOTIFY_DEADSTATE - один или несколько файлов в плейлисте были помечены как "not found"
  • AIMP_PLAYLIST_NOTIFY_MAKEVISIBLE - пользователь запросил сделать проигрываемый файл видимым
  • AIMP_PLAYLIST_NOTIFY_PLAYBACKQUEUE - изменился порядок воспроизведения записей в плейлисте
  • AIMP_PLAYLIST_NOTIFY_PLAYBACKCURSOR - изменился курсор воспроизведения
  • AIMP_PLAYLIST_NOTIFY_PLAYBACKBOOKMARK - изменилась временная метка курсора воспроизведения (v5.40)
  • AIMP_PLAYLIST_NOTIFY_FOCUSINDEX - фокус перешел на другую запись
  • AIMP_PLAYLIST_NOTIFY_CONTENT - изменилось порядок записей и/или групп в плейлисте
  • AIMP_PLAYLIST_NOTIFY_FILEINFO - изменилась информация в одной или нескольких записях
  • AIMP_PLAYLIST_NOTIFY_STATISTICS - изменилась статистика плейлиста (кол-во файлов, длительность, размер)
  • AIMP_PLAYLIST_NOTIFY_PLAYINGSWITCHS - изменилось состояние переключателей авто воспроизведения у одной или нескольких записей
  • AIMP_PLAYLIST_NOTIFY_READONLY - изменился статус "только чтение"
  • AIMP_PLAYLIST_NOTIFY_PREIMAGE - изменился путь к прообразу
Removed
Событие возникает перед удалением / выгрузкой плейлиста

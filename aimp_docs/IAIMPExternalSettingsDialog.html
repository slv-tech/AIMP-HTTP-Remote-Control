IAIMPExternalSettingsDialog
Описание
Если плагин имеет внешний диалог настроек, который не интегрируется в программу с помощью Options Dialog API, он должен реализовывать данный интерфейс на ряду с IAIMPPlugin.
Методы
Show
Метод вызова модального диалога с настройками.
inParentWindowHWNDДескриптор окна, что вызывает диалог

Резюме

Объявление:apiPlugin.pas / apiPlugin.h
Асинхронная работа:Не поддерживается
Приложения:Player, Audio Converter, Advanced Tag Editor

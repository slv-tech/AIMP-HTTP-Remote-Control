AIMP_RA_CMD_GET_ALBUMART
AIMP_RA_CMD_GET_ALBUMART
Команда запроса обложки альбома для проигрываемого трека.

Выполняется в два этапа:
1 - программа посылает запрос на получение обложки альбома
2 - плеер отправляет ответ указанному в Param окну в виде WM_COPYDATA

LParam сообщения WM_COPYDATA содержит указатель на структуру CopyDataStruct, которая будет содержать следующие данные:
  • dwData - ключевое слово, должно быть = WM_AIMP_COPYDATA_ALBUMART_ID
  • lpData - указатель на первый элемент массива байтов, который содержит обложку альбома в формате PNG
  • cbData - количество элементов в массиве
inParamLPARAM Дескриптор окна-получателя
outResultLRESULT Возвращает 0, если плеер ничего не проигрывает или трек не имеет обложки альбома

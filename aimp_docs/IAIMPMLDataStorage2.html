IAIMPMLDataStorage2
Описание
Расширенная версия интерфейса IAIMPMLDataStorage, позволяющая создавать объекты привязанные к хранилищу данных.

На этом же уровне реализуются провайдеры данных - IAIMPMLDataProvider, IAIMPMLDataProvider2. Вы можете получить доступ к ним через метод IAIMPMLDataStorage.QueryInterface(...).
Предок
IAIMPMLDataStorage2 наследуется от IAIMPMLDataStorage
Методы
CreateObject
inIDTGUID Идентификатор интерфейса объекта.

На текущий момент поддерживаются:
  • IID_IAIMPMLDataFilter - идентификатор объекта IAIMPMLDataFilter
    • outObjUntyped Ссылка на объект
    outResultHRESULTS_OK - Метод успешно завершился
    E_INVALIDARG - Один из параметров задан неверно
    E_NOTIMPL - Метод не реализован
    E_UNEXPECTED - Произошел непредвиденный сбой
    E_NOINTERFACE - если объект не имеет интерфейса с указанным IID.
    Создание объекта
    <IAIMPServiceMusicLibrary>.GetStorageByID(.., IID_IAIMPMLDataStorage2, ..)

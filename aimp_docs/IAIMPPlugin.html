IAIMPPlugin
Методы
InfoGet
Метод возвращает информацию о плагине.
Не должно требовать инициализации плагина.
inIndexInteger Один из флагов:
  • AIMP_PLUGIN_INFO_NAME
  • AIMP_PLUGIN_INFO_AUTHOR
  • AIMP_PLUGIN_INFO_SHORT_DESCRIPTION
  • AIMP_PLUGIN_INFO_FULL_DESCRIPTION
outResultPChar Возвращает информацию по запрашиваемому Index-у. Может возвращать nil (null).
InfoGetCategories
Метод указывает, к каким категориям относится плагин.
Эта информация используется менеджером плагинов для только сортировки плагинов в списке.
Не должно требовать инициализации плагина.
outResultLongWord Возвращает один или комбинацию из следующих флагов:
  • AIMP_PLUGIN_CATEGORY_ADDONS
  • AIMP_PLUGIN_CATEGORY_DECODERS
  • AIMP_PLUGIN_CATEGORY_VISUALS
  • AIMP_PLUGIN_CATEGORY_DSP
Initialize
inCoreIAIMPCore Ссылка на интерфейс доступа к ядру программы.
outResultHRESULTОбратите внимание, что плагин может быть загружен не только плеером, но и утилитой. Поэтому при инициализации плагина стоит проверить, есть ли в текущем окружении все необходимые встроенные сервисы. В случае отсутствия функционально необходимых сервисов - плагину рекомендуется вернуть код ошибки - в этом случае программа пометит плагин как неиспользуемый.

S_OK - в случае успеха, в случае ошибки - любое другое значение.

Внимание! Метод Finalize будет вызван только в том случае, если эта функция вернет S_OK.

См. рекомендации по работе с сервисами.
Finalize
Метод финализации плагина. Вызывается перед выгрузкой DLL-ки, если метод Initialize вернул S_OK.
SystemNotification
Обработчик нотификаций изменений в ядре программы.
См. рекомендации по работе с сервисами
inNotifyIDInteger Идентификатор нотификации.
Может быть одним из значений:
  • AIMP_SYSTEM_NOTIFICATION_SERVICE_ADDED
  • AIMP_SYSTEM_NOTIFICATION_SERVICE_REMOVED
  • AIMP_SYSTEM_NOTIFICATION_EXTENSION_REMOVED
inDataIUnknown Изменяемый объект. Может быть равен nil (null);
Примечание
Если плагин имеет внешний диалог настроек, он должен реализовывать интерфейс IAIMPExternalSettingsDialog на ряду с IAIMPPlugin.
Создание экземпляра
Экземпляр создается плагином и передается программе при вызове экспортируемой функции AIMPPluginGetHeader.

function AIMPPluginGetHeader(out Header: IAIMPPlugin): HRESULT; stdcall;
Описание интерфейса
Описание интерфейса расположено в apiPlugin.pas / apiPlugin.h

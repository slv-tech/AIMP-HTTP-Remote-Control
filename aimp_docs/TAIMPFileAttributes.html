TAIMPFileAttributes
Описание
Структура содержит атрибуты и время создания/изменения файла

TAIMPFileAttributes
inAttributesLongWord Соответствует атрибутам WIN32_FIND_DATA
inTimeCreationTDateTime Дата и время создания файла.

Целая часть - дни, дробная часть - миллисекунды.
Формат времени совместим с типом TDateTime в Delphi.
inTimeLastAccessTDateTime Дата и время последнего доступа к файлу.

Целая часть - дни, дробная часть - миллисекунды.
Формат времени совместим с типом TDateTime в Delphi.
inTimeLastWriteTDateTime Дата и время изменения файла.

Целая часть - дни, дробная часть - миллисекунды.
Формат времени совместим с типом TDateTime в Delphi.
inReserved0Int64 Для будущего использования, должно быть равно 0
inReserved1Int64 Для будущего использования, должно быть равно 0
inReserved2Int64 Для будущего использования, должно быть равно 0

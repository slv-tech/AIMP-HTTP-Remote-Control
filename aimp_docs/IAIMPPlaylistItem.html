IAIMPPlaylistItem
Описание
Интерфейс IAIMPPlaylistItem представляет собой хранилище расширенной информации о записи в плейлисте.
Предок
IAIMPPlaylistItem наследуется от IAIMPPropertyList.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMP_PLAYLISTITEM_PROPID_CUSTOM
Пользовательское однострочное текстовое поле, может задаваться плагином. Значение поля сохраняется в плейлист.

Текст не должен содержать символов переноса строки, а так же "|".

Вы можете использовать значение этого поля для сортировки / группировки записей в плейлисте с помощью макроса %Custom
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_PLAYLISTITEM_PROPID_DISPLAYTEXT
Отображаемая в плейлисте строка, формат определяется текущими настройками плейлиста.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPString, ..)
AIMP_PLAYLISTITEM_PROPID_FILEINFO
ТипIAIMPFileInfo
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPFileInfo, ..)
ЗаписьОсуществляется путем модификации свойств полученного <IAIMPFileInfo>
AIMP_PLAYLISTITEM_PROPID_FILENAME
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_PLAYLISTITEM_PROPID_GROUP
ТипIAIMPPlaylistGroup
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPPlaylistGroup, ..)
AIMP_PLAYLISTITEM_PROPID_INDEX
Абсолютная позиция записи в плейлисте
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_PLAYLISTITEM_PROPID_MARK
Оценка записи, от 0.0 до 5.0
ТипDouble
ЧтениеIAIMPPropertyList.GetValueAsFloat
ЗаписьIAIMPPropertyList.SetValueAsFloat
AIMP_PLAYLISTITEM_PROPID_PLAYBACKQUEUEINDEX
Порядковый номер записи в очереди воспроизведения.
Может не совпадать с порядковым номером записи в плейлисте!
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
AIMP_PLAYLISTITEM_PROPID_PLAYINGSWITCH
Состояние переключателя автоматического проигрывания, 0 - выключен, любое другое - включен.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32

ЗаписьIAIMPPropertyList.SetValueAsObject(.., ..)
AIMP_PLAYLISTITEM_PROPID_SELECTED
Значение отличное от нуля - запись выбрана.
ТипInteger

E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

IAIMPServiceVersionInfo
Описание
Этот сервис предоставляет доступ к информации о версии программы.
Методы
FormatInfo
Метод возвращает информацию о версии в формате 
v<мажорный номер>.<минорный номер> <статус сборки>, build <номер сборки> (<дата сборки>)
Например:
v3.60, build 1400 (18.12.2013)
v3.60 Beta, build 1400 (18.12.2013)
outSIAIMPStringРезультирующая строка
outResultHRESULT S_OK - Метод успешно завершился
E_UNEXPECTED - Произошел непредвиденный сбой
GetBuildDate
outResultInteger Дата сборки.
HiWord(Result) - год
HiByte(LoWord(Result)) - месяц
LoByte(LoWord(Result)) - день
GetBuildState
outResultInteger Результат функции указывает на стадию разработки:
  • AIMP_SERVICE_VERSION_STATE_RELEASE
  • AIMP_SERVICE_VERSION_STATE_RELEASE_CANDIDATE
  • AIMP_SERVICE_VERSION_STATE_BETA
  • AIMP_SERVICE_VERSION_STATE_ALPHA
Примечание: флаги размещены в порядке убывания готовности продукта.
GetBuildNumber
outResultInteger Абсолютный номер сборки, не зависит от версии.
GetVersionID
outResultIntegerID версии в формат XYYZ.
X - мажорная версия, не ограничена по количеству разрядов
YY - минорная версия, ограничена двумя разрядами
Z - номер редакции, ограничен одним разрядом

Примеры:
3551 -> 3.55.1
10051 -> 10.05.1
0010 -> 0.01.0

Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceVersionInfo, ...);

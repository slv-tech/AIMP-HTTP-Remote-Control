IAIMPExtensionFileExpander
Описание
IAIMPExtensionFileExpander представляет собой расширение для сервиса IAIMPServiceFileManager, позволяющее разделить реальный файл-контейнер на несколько виртуальных файлов.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
Expand
Метод разделяет реальный файл-контейнер на несколько виртуальных файлов.
inFileNameIAIMPString Имя реального файла
outListIAIMPObjectList Список виртуальных файлов (см. IAIMPVirtualFile)
Список должен создаваться и заполняться только в том случае, если результат выполнения метода равен S_OK.
inProgressCallbackIAIMPProgressCallback Функция обратного вызова для возможности отображения прогресса разделения файла-контейнера.
Может быть равен nil (null).
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

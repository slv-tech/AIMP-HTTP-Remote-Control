IAIMPFileStream
Описание
IAIMPFileStream представляет собой вариант IAIMPStream, где все данные хранятся в файле на жестком диске.
Предок
IAIMPFileStream наследуется от IAIMPStream
Методы
GetClipping
В случае, если поток создан для участка файла, метод возвращает границы этого участка.
Cм. IAIMPServiceFileStreaming.CreateStreamForFile
outOffsetInt64
outSizeInt64
outResultHRESULT S_OK - Метод успешно завершился
E_FAIL - Файл не ограничен
E_UNEXPECTED - Произошел непредвиденный сбой
GetFileName
outSIAIMPString Имя реального файла на жестком диске.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой

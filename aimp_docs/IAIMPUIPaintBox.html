IAIMPUIPaintBox
Описание
IAIMPUIPaintBox является наследником базового элемента управления IAIMPUIControl, позволяющий полностью переопределить его отрисовку.
События
IAIMPUIDrawEvents
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

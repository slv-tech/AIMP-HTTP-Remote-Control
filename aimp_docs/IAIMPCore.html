IAIMPCore
Описание
IAIMPCore - является основным связующим звеном между плагином и программой. Через этот интерфейс осуществляется доступ к сервисам, а так же регистрация собственных сервисов и расширений.
Методы
CreateObject
Метод служит для создания примитивных объектов
inIIDTGUIDИдентификатор интерфейса объекта
outObjUntypedОбъект
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_NOINTERFACE - если объект не имеет интерфейса с указанным IID.
GetPath
Метод возвращает путь к одной из "системных" папок плеера.
inPathIDInteger Идентификатор папки, может иметь одно из следующих значений:
  • AIMP_CORE_PATH_AUDIOLIBRARY - путь к папке, где хранится база данных фонотеки текущего пользователя
  • AIMP_CORE_PATH_HELP - путь к папке, где хранятся руководства
  • AIMP_CORE_PATH_ICONS - путь к папке, где хранятся наборы иконок
  • AIMP_CORE_PATH_LANGS - путь к папке, где хранятся файлы локализаций
  • AIMP_CORE_PATH_PLAYLISTS - путь к папке, где хранятся плейлисты текущего пользователя
  • AIMP_CORE_PATH_PLUGINS - путь к папке с плагинами
  • AIMP_CORE_PATH_PROFILE - путь к папке профиля текущего пользователя
  • AIMP_CORE_PATH_SKINS - путь к папке, где хранятся скины текущего пользователя
  • AIMP_CORE_PATH_SKINS_COMMON - путь к общей (для всех пользователей) папке, где хранятся скины
outValueIAIMPStringПуть к папке
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
RegisterExtension
Метод регистрирует предоставляемые плагином расширения в программе.
Доступ к расширениям осуществляется через соответствующие для этого типа расширений сервисы. Например: IAIMPServiceAudioDecoders

Расширения автоматически удаляются из ядра перед финализацией плагина.
inServiceIIDTGUIDИдентификатор сервиса, в который нужно зарегистрировать расширение
inExtensionIUnknownРегистрируемое расширение
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
E_NOINTERFACE - сервис с указанным IID не зарегистрирован в ядре программы или не поддерживает расширения.
RegisterService
Метод регистрирует предоставляемые плагином сервисы в программе.
Сервисы автоматически удаляются из ядра перед финализацией плагина.
inServiceIUnknownРегистрируемый сервис
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
UnregisterExtension
Метод позволяет вручную снять с регистрации указанное расширение.

Замечание 1: расширения автоматически удаляются из ядра перед финализацией плагина, поэтому вызывать вручную этот метод не обязательно
Замечание 2: расширение может использоваться в текущий момент времени, в таком случае оно будет полностью снято с регистрации только освобождения
inExtensionIUnknownРасширение
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Создание экземпляра
Экземпляр создается программой и передается плагину при инициализации.

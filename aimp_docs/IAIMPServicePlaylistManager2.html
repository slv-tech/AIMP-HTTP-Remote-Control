IAIMPServicePlaylistManager2
Описание
Интерфейс предоставляет доступ ко всем зарегистрированным IAIMPExtensionPlaylistPreimageFactory
Предок
IAIMPServicePlaylistManager2 наследуется от IAIMPServicePlaylistManager.
Методы
GetPreimageFactory
Метод возвращает экземпляр интерфейса фабрики по ее индексу.
inIndexInteger Индекс плейлиста, от 0 до GetPreimageFactoryCount - 1.
outFactoryIAIMPExtensionPlaylistPreimageFactory
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetPreimageFactoryByID
Метод возвращает экземпляр интерфейса фабрики по ее ID.
inIDIAIMPString
outFactoryIAIMPExtensionPlaylistPreimageFactory
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetPreimageFactoryCount
outResultInteger Количество зарегистрированных фабрик
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServicePlaylistManager2, ...);

IAIMPServiceAlbumArtCache
Описание
Сервис предоставляет доступ к кэш-менеджеру обложек альбома и позволяет сбросить кэш по одному из критериев.
Методы
Flush
Метод позволяет сбросить весь кэш.

Внимание! Частый сброс кэша может привести к ухудшению производительности программы.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Get
Возвращает IAIMPImageContainer для указанного ключа.
Обратите внимание, что несмотря на успешное выполнение функции, Image может быть равен nil / null.
inKeyIAIMPStringСтрока-ключ должна быть построена так, чтобы однозначно идентифицировать обложку альбома для указанных IAIMPFileInfo.
varImageIAIMPImageContainer
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Put
Оптимизирует IAIMPImageContainer и записывает его в кэш.
В качестве Image может быть передан nil / null, если нужно закэшировать "отсутствующее значение".
inKeyIAIMPStringСтрока-ключ должна быть построена так, чтобы однозначно идентифицировать обложку альбома для указанных IAIMPFileInfo.

Remove
Сбрасывает кэш для указанного ключа.

IAIMPMLDataFieldFilter
Описание
Фильтр проверяет соответствие значения поля указанному правилу.
Предок
IAIMPMLDataFieldFilter наследуется от IAIMPPropertyList2
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPML_FIELDFILTER_FIELD
Имя колонки, по которой производится фильтрация.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPML_FIELDFILTER_OPERATION
Может принимать одно из значений:
  • AIMPML_FIELDFILTER_OPERATION_EQUALS
  • AIMPML_FIELDFILTER_OPERATION_NOTEQUALS
  • AIMPML_FIELDFILTER_OPERATION_BETWEEN (только для чисел)
  • AIMPML_FIELDFILTER_OPERATION_LESSTHAN (только для чисел)
  • AIMPML_FIELDFILTER_OPERATION_LESSTHANOREQUALS (только для чисел)
  • AIMPML_FIELDFILTER_OPERATION_GREATERTHAN (только для чисел)
  • AIMPML_FIELDFILTER_OPERATION_GREATERTHANOREQUALS (только для чисел)
  • AIMPML_FIELDFILTER_OPERATION_CONTAINS (только для строк)
  • AIMPML_FIELDFILTER_OPERATION_BEGINSWITH (только для строк)
  • AIMPML_FIELDFILTER_OPERATION_ENDSWITH (только для строк)
  • AIMPML_FIELDFILTER_OPERATION_ISLASTXDAYS (только для даты и времени)

Операции со строками не должны учитывать регистр.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPML_FIELDFILTER_VALUE1
Значение 1
ТипOleVariant
ЧтениеIAIMPPropertyList.GetValueAsVariant(ID, ..)
ЗаписьIAIMPPropertyList.SetValueAsVariant(ID, ..)
AIMPML_FIELDFILTER_VALUE2
Значение 2. Используется только при AIMPML_FIELDFILTER_OPERATION = AIMPML_FIELDFILTER_OPERATION_BETWEEN.
ТипOleVariant
ЧтениеIAIMPPropertyList.GetValueAsVariant(ID, ..)
ЗаписьIAIMPPropertyList.SetValueAsVariant(ID, ..)

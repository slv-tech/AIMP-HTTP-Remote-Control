IAIMPUIBaseEdit
Описание
IAIMPUIBaseEdit представляет собой базовый интерфейс для всех полей ввода.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIBaseEdit
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_BASEEDIT_PROPID_BORDERS
0 - скрыть рамку, 1 - показать.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BASEEDIT_PROPID_MAXLENGTH
Ограничение на длину вводимого текста.
0 - не ограничено.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BASEEDIT_PROPID_READONLY
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BASEEDIT_PROPID_SELSTART
Определяет позицию первого выбранного символа, где 0 - первый символ. Если ничего не выбрано - определяет позицию курсора в тексте.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BASEEDIT_PROPID_SELLENGTH
Количество выбранных символов.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BASEEDIT_PROPID_SELTEXT
Выбранный текст.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_BASEEDIT_PROPID_TEXT
Весь текст.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)

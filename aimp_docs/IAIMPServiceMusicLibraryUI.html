IAIMPServiceMusicLibraryUI
Описание
IAIMPServiceMusicLibraryUI сервис предоставляет доступ к данным из UI-элементов для работы с активным хранилищем данных, с которыми взаимодействует пользователь.
Методы
GetFiles
Метод возвращает список файлов из активного хранилища данных
inFlagsLongWord Один из флагов:
AIMPML_GETFILES_FLAGS_ALL
AIMPML_GETFILES_FLAGS_SELECTED
AIMPML_GETFILES_FLAGS_FOCUSED
outListIAIMPMLFileList
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

Метод позволяет перевести фокус в дереве группировки на другой узел.

IAIMPServiceFileManager
Описание
IAIMPServiceFileManager не имеет методов, интерфейс введен для возможности регистрации пользовательских расширений.

Пример использования:
  Сore.RegisterExtension(IID_IAIMPServiceFileManager, ...);
Резюме
Объявление:apiFileManager.pas / apiFileManager.h
Асинхронная работа:Поддерживается
Расширения:IAIMPExtensionFileExpander
Приложения:Player, Audio Converter, Advanced Tag Editor
Минимальная версия:3.60.1455

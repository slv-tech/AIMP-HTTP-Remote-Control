IAIMPHTTPClientEvents2
Описание
Интерфейс является расширением для IAIMPHTTPClientEvents. Должен быть реализован плагином на том же уровне, что и IAIMPHTTPClientEvents.

Интерфейс позволяет плагину получить "сырой" заголовок ответа сервера.
Методы
OnAcceptHeaders
Событие возникает при получении HTTP-заголовка от сервера
inHeaderIAIMPString"Сырой" HTTP-заголовок ответа от сервера
varAllowLongBool Плагин может отказаться принимать данные, для этого необходимо выставить данный параметр в FALSE

TAIMPPlaylistDeleteProc
Описание
TAIMPPlaylistDeleteProc - функция обратного вызова для отбора записей на удаление.
Функция позволяет плагину реализовать свое условие удаления записей из плейлиста.

Функция должна поддерживать асинхронную работу!

inItem1IAIMPPlaylistItem
inUserDataPointer Пользовательский параметр, передается плагином вместе с функцией в вызываемый метод
outResultLongBool True - если запись должна быть удалена из плейлиста

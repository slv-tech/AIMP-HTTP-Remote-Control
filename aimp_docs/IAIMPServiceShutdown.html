IAIMPServiceShutdown
Описание
IAIMPServiceShutdown позволяет завершить работу приложения и/или Windows.
Методы
Restart
Метод позволяет перезапустить приложение.
inParamsIAIMPString Параметры командной строки, может быть равен nil / null
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Shutdown
Метод позволяет завершить работу приложения и/или Windows
inFlagsLongWord Комбинация из флагов:
  • AIMP_SERVICE_SHUTDOWN_FLAGS_HIBERNATE - Перевести компьютер в гибернацию
  • AIMP_SERVICE_SHUTDOWN_FLAGS_POWEROFF - Выключить компьютер
  • AIMP_SERVICE_SHUTDOWN_FLAGS_SLEEP - Перевести компьютер в спящий режим
  • AIMP_SERVICE_SHUTDOWN_FLAGS_LOGOFF - Выйти из текущего пользователя
  • AIMP_SERVICE_SHUTDOWN_FLAGS_REBOOT - Перезагрузить компьютер
  • AIMP_SERVICE_SHUTDOWN_FLAGS_CLOSE_APP - Закрыть приложение
  • AIMP_SERVICE_SHUTDOWN_FLAGS_NO_CONFIRM - Подавляет показ диалога с подтверждением завершения работы компьютера

Примечание 1: флаги AIMP_SERVICE_SHUTDOWN_FLAGS_HIBERNATE, AIMP_SERVICE_SHUTDOWN_FLAGS_SHUTDOWN, AIMP_SERVICE_SHUTDOWN_FLAGS_LOGOFF, AIMP_SERVICE_SHUTDOWN_FLAGS_REBOOT и AIMP_SERVICE_SHUTDOWN_FLAGS_SLEEP не должны использоваться одновременно.

Примечание 2: Отсутствие флага AIMP_SERVICE_SHUTDOWN_FLAGS_CLOSE_APP учитывается только в случае перевода компьютера в спящий режим или режим гибернации.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceShutdown, ...);

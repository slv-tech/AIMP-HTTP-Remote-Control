IAIMPUIInputDialog
Описание
IAIMPUIInputDialog предоставляет доступ диалогу ввода данных. Диалог поддерживает ввод текста, целых и дробных чисел, а так же возможность ввода нескольких значений сразу. Тип поля ввода определяется по типу значения, что передано в функцию вызова.
Методы
Execute
Вызывает диалог для ввода одного значения
inOwnerWndHWND Дескриптор окна-владельца диалога.
Если указать 0 - будет использоваться дескриптор главного окна.
inCaptionIAIMPString
inEventsHandlerIUnknown см. IAIMPUIInputDialogEvents
inTextIAIMPString Текст, что будет отображаться рядом с полем ввода
varValueOleVariant
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Execute2
Вызывает диалог для ввода нескольких значений
inOwnerWndHWND Дескриптор окна-владельца диалога.
Если указать 0 - будет использоваться дескриптор главного окна.
inCaptionIAIMPString
inEventsHandlerIUnknown см. IAIMPUIInputDialogEvents
inTextForValuesIAIMPObjectList Текст, что будет отображаться рядом с полем ввода. Количество объектов в списке должно совпадать с ValueCount.
inValuesPOleVariant Указатель на первый элемент массива OleVariant-элементов
inValueCountInteger Количество элементов в массиве
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Доступ к интерфейсу
<IAIMPServiceUI>.QueryInterface(IID_IAIMPUIInputDialog, <IAIMPUIInputDialog>)

TAIMPServiceLyricsReceiveProc
Описание
TAIMPServiceLyricsReceiveProc - функция обратного вызова для получения текста песни в результате асинхронного запроса. Функция вызывается главным потоком программы.
Методы
inLyricsIAIMPLyrics В случае, если текст песни не найден - Lyrics будет равен nil / null
inUserDataPointer Пользовательский параметр, передается плагином вместе с функцией в вызываемый метод
Резюме
Объявление:apiLyrics.pas / apiLyrics.h
Асинхронная работа:Поддерживается
Приложения:Player, Audio Converter, Advanced Tag Editor

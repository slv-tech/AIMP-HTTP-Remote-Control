Соглашение имен свойств
Шаблон имени идентификатора свойства
Имя константы идентификатора свойства строится по определенному правилу и содержит в себе дополнительную информацию: тип объекта и описание свойства.

Формат имени выглядит следующим образом:
    AIMP_OOOOO_PROPID_DDDDD
где:
    OOOOO - тип объекта-предка (без префикса "IAIMP")
    DDDDD - описание свойства

Пример 1:
    AIMP_FILEINFO_PROPID_ALBUMGAIN

    Описание свойства: AlbumGain
    Тип объекта: FILEINFO - IAIMPFileInfo

Пример 2:
    AIMP_FILEINFO_PROPID_ALBUMARTIST

    Описание свойства: AlbumArtist
    Тип объекта: FILEINFO - IAIMPFileInfo

Пользовательские идентификаторы
Пользовательские идентификаторы обязаны начинаться с AIMP_PROPERTYLIST_CUSTOM_PROPID_BASE во избежании конфликтов с настоящей и будущими версиями API.

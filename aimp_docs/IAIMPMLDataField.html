IAIMPMLDataField
Описание
Представляет описание поля движку фонотеки.
Предок
IAIMPMLDataField наследуется от IAIMPPropertyList
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPML_FIELD_PROPID_CUSTOM
Пользовательский параметр, позволяет прикрепить к экземпляру интерфейса пользовательский набор данных
ТипIUnknown
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IUnknown, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IUnknown>)
AIMPML_FIELD_PROPID_NAME
Внутреннее имя поля, используется как уникальный идентификатор в пределах текущего хранилища данных.

Отображаемое имя берется из пользовательской локализации из секции AIMPML (AIMPML\<FieldName>).
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPML_FIELD_PROPID_TYPE
Тип поля, одно из значений:
  • AIMPML_FIELDTYPE_INT32
  • AIMPML_FIELDTYPE_INT64
  • AIMPML_FIELDTYPE_FLOAT
  • AIMPML_FIELDTYPE_STRING
  • AIMPML_FIELDTYPE_STRINGSET - набор строковых значений, отделенных друг от друга символом ";"
  • AIMPML_FIELDTYPE_DATETIME - тип: AIMPML_FIELDTYPE_FLOAT, в UI отображается форматированное значение (формат может выбираться в зависимости от контекста)
  • AIMPML_FIELDTYPE_DURATION - тип: AIMPML_FIELDTYPE_FLOAT, в UI отображается форматированное значение (формат может выбираться в зависимости от контекста)
  • AIMPML_FIELDTYPE_FILESIZE - тип: AIMPML_FIELDTYPE_INT64, в UI отображается форматированное значение (формат может выбираться в зависимости от контекста)
  • AIMPML_FIELDTYPE_FILENAME - тип: AIMPML_FIELDTYPE_STRING, в UI: разбивает строку на уровни в дереве группировки по символам "\" и "/". Группировка по этому полю совместно с другими полями недоступна.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPML_FIELD_PROPID_FLAGS
0 или комбинация флагов:
  • AIMPML_FIELDFLAG_GROUPING - разрешает пользователю группировать по этому полю
  • AIMPML_FIELDFLAG_FILTERING - разрешает пользователю фильтровать по этому полю
  • AIMPML_FIELDFLAG_INTERNAL - помечает поле внутренним: оно не показывается пользователю
  • AIMPML_FIELDFLAG_REQUIRED - помечает поле обязательным: оно всегда участвует в выборке данных
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPML_FIELD_PROPID_IMAGE
Номер картинки в дереве группировки, что будет отображаться на уровне с этим полем:
  • AIMPML_FIELDIMAGE_FOLDER
  • AIMPML_FIELDIMAGE_ARTIST
  • AIMPML_FIELDIMAGE_DISK
  • AIMPML_FIELDIMAGE_NOTE
  • AIMPML_FIELDIMAGE_STAR
  • AIMPML_FIELDIMAGE_CALENDAR
  • AIMPML_FIELDIMAGE_LABEL
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPML_FIELD_PROPID_DISPLAYVALUE
Позволяет назначить обработчик для переопределения отображаемого значения поля.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
ТипIAIMPMLDataFieldDisplayValue
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPMLDataFieldDisplayValue, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPMLDataFieldDisplayValue>)
Создание объекта
<IAIMPCore>.CreateObject(IID_IAIMPMLDataField, ...)

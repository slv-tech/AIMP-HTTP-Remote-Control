IAIMPMLDataStorageManager
Описание
Интерфейс предоставляет плагину возможность уведомлять фонотеку об изменениях в своем хранилище данных. Передается в плагин после успешной регистрации IAIMPMLExtensionDataStorage при вызове метода IAIMPMLExtensionDataStorage.Initialize.
Методы
BackgroundTaskStarted
Связка BackgroundTaskStarted / BackgroundTaskFinished позволяет плагину сообщить движку о работающих фоновых задачах.

Примечание 1: первая активная фоновая задача отображается пользователю (эта возможность зависит от используемого скина).
Примечание 2: обе функции должны вызываться из главного потока.
inIDInteger Уникальный идентификатор фоновой задачи.
inCaptionIAIMPString Текст, что будет виден пользователю.

Вы можете менять содержимое переданного в функцию экземпляра строки во время выполнения фоновой операции, изменения будут отображены на экране после небольшой задержки.
inCancelEventIAIMPActionEvent Обработчик события, позволяющего отменить выполнение фоновой операции.
Может принимать nil/null - в этом случае, пользователю не будет доступна возможность отмены операции.
BackgroundTaskFinished
inIDInteger Уникальный идентификатор фоновой задачи.
Changed
Сообщает движку, что нужно полностью перегрузить данные для зависимых смарт-плейлистов и фонотеки.

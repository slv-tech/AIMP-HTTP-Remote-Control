IAIMPMLDataFilterGroup
Описание
Группа позволяет объединить результаты вычисления нескольких дочерних правил
Предок
IAIMPMLDataFilterGroup наследуется от IAIMPPropertyList2
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPML_FILTERGROUP_OPERATION
Правило обработки результатов от дочерних фильтров.
Может принимать одно из значений:
  • AIMPML_FILTERGROUP_OPERATION_OR - результат = True, если один из дочерних фильтров вернул True
  • AIMPML_FILTERGROUP_OPERATION_AND - результат = True, если все дочерние фильтры вернули True
  • AIMPML_FILTERGROUP_OPERATION_NOTOR - результат = True, если все дочерние фильтры вернули False
  • AIMPML_FILTERGROUP_OPERATION_NOTAND - результат = True, если один из дочерних фильтров вернул False
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32

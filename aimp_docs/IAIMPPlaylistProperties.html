IAIMPPlaylistProperties
Description
Интерфейс расширяет IAIMPPropertyList2 и предоставляет доступ к пользовательским свойствам плейлиста, которые будут сохраняться в файл плейлиста (для форматов AIMPPL4 и XSPF).

Доступ к IAIMPPlaylistProperties осуществляется через <IAIMPPlaylist>.QueryInterface(IID_IAIMPPlaylistProperties, ..);
Methods
GetCustomValue
inNameIAIMPStringИмя свойства
outValueIAIMPStringЗначение свойства, может быть многострочным
outResultHRESULT S_OK - вызов успешно завершен.
E_FAIL - свойство с таким именем отсутствует у плейлиста.
E_ACCESSDENIED - вызов совершен не из главного потока.
SetCustomValue
inNameIAIMPStringИмя свойства
inValueIAIMPStringЗначение свойства, может быть многострочным
Чтобы удалить свойство, укажите nil/null в качестве значения Value.
outResultHRESULT S_OK - вызов успешно завершен.
E_ACCESSDENIED - вызов совершен не из главного потока.

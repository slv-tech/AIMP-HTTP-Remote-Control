IAIMPUIButton
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIButton
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_BUTTON_PROPID_CAPTION
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_BUTTON_PROPID_FOCUSABLE
Опция показывает, можно ли фокусировать кнопку мышкой или с клавиатуры.
0 - фокусирование отключено. По умолчанию: 1.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BUTTON_PROPID_DEFAULT
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BUTTON_PROPID_DROPDOWNMENU
Выпадающее меню, которое показывается при клике ЛКМ по кнопке.
Актуально для AIMPUI_BUTTON_PROPID_STYLE = AIMPUI_FLAGS_BUTTON_STYLE_DROPDOWN или AIMPUI_FLAGS_BUTTON_STYLE_DROPDOWNBUTTON.
ТипIAIMPUIPopupMenu
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPUIPopupMenu, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPUIPopupMenu>)
AIMPUI_BUTTON_PROPID_IMAGEINDEX
Индекс картинки в IAIMPUIImageList для отрисовки.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BUTTON_PROPID_IMAGELIST
ТипIAIMPUIImageList
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPUIImageList, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPUIImageList>)
AIMPUI_BUTTON_PROPID_MODALRESULT
Актуально при использовании в модальных диалогах. Клик по кнопке с ModalResult отличным от NONE закроет модальную форму, и выставит результат ее вызова в значение этого свойства.

Может принимать одно из значений:

+ AIMPUI_FLAGS_MODALRESULT_NONE
+ AIMPUI_FLAGS_MODALRESULT_OK
+ AIMPUI_FLAGS_MODALRESULT_CANCEL
+ AIMPUI_FLAGS_MODALRESULT_ABORT
+ AIMPUI_FLAGS_MODALRESULT_RETRY
+ AIMPUI_FLAGS_MODALRESULT_IGNORE
+ AIMPUI_FLAGS_MODALRESULT_YES
+ AIMPUI_FLAGS_MODALRESULT_NO
+ AIMPUI_FLAGS_MODALRESULT_CLOSE
+ AIMPUI_FLAGS_MODALRESULT_HELP
+ AIMPUI_FLAGS_MODALRESULT_TRYAGAIN
+ AIMPUI_FLAGS_MODALRESULT_CONTINUE

ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BUTTON_PROPID_STYLE
Стиль кнопки, может принимать одно из значений:



+ AIMPUI_FLAGS_BUTTON_STYLE_NORMAL - обычная кнопка
+ AIMPUI_FLAGS_BUTTON_STYLE_DROPDOWN - кнопка с меню, при клике показывается DropDownMenu
+ AIMPUI_FLAGS_BUTTON_STYLE_DROPDOWNBUTTON - гибрид первых двух стилей: левая часть ведет себя как обычная кнопка, правая - как DropDown - показывает DropDownMenu

ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BUTTON_PROPID_TEXTSTYLE
Начертание шрифта.
0 или комбинация из следующих флагов:

+ AIMPUI_FLAGS_FONT_BOLD
+ AIMPUI_FLAGS_FONT_ITALIC
+ AIMPUI_FLAGS_FONT_STRIKEOUT
+ AIMPUI_FLAGS_FONT_UNDERLINE

ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32

Методы

outResultHRESULTS_OK - Метод успешно завершился

События
IAIMPUIChangeEvents вызывается при клике по кнопке
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

IAIMPMLDataFieldFilterByArray
Описание
Фильтр проверяет наличие значения поля среди массива заранее заданных значений.
Является аналогом выражения: 
(FieldValue = Value1) OR (FieldValue = Value2) OR ... OR (FieldValue = ValueN)
Предок
IAIMPMLDataFieldFilterByArray наследуется от IAIMPPropertyList2
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPML_FIELDFILTERBYARRAY_FIELD
Имя колонки, по которой производится фильтрация.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)

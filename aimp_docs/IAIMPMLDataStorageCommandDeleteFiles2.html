IAIMPMLDataStorageCommandDeleteFiles2
Описание
Общий случай команды IAIMPMLDataStorageCommandDeleteFiles, позволяет удалять записи по указанному фильтру (хранилище данных должно поддерживать фильтрацию данных на своей стороне, см. AIMPML_DATASTORAGE_PROPID_CAPABILITIES)
Предок
IAIMPMLDataStorageCommandDeleteFiles2 наследуется от IAIMPMLDataStorageCommandDeleteFiles
Методы
Delete2
inFilterIAIMPMLDataFilter
inPhysicallyLongBool
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.

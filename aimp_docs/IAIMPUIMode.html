IAIMPUIMode
Описание
IAIMPUIMode предоставляет доступ к настройкам отображения пользовательского интерфейса.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_MODE_PROPID_ACCENT
Цвет акцента приложения в формате ARGB, 0 - акцент не задан.
Для темирования рекомендуется использовать IAIMPUIColorSchema.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
AIMPUI_MODE_PROPID_DPI
Пользовательский масштаб UI приложения, в dpi (96 dpi = 100%).
0 - значение пользователем не задано и должно определяться автоматически.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
AIMPUI_MODE_PROPID_STYLE
Стиль отображения: дневной / ночной. Если в настройках стоит "автоматически" - плеер сам вычислит актуальное значение стиля.
Может принимать одно из следующих значений: AIMPUI_STYLE_LIGHT / AIMPUI_STYLE_DARK.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32

Получение экземпляра
Получить ссылку на интерфейс можно с помощью <IAIMPServiceUI>.QueryInterface.

Уведомления
В момент изменения настроек отображения UI делается широковещательная рассылка сообщения AIMP_MSG_EVENT_UI_MODE.

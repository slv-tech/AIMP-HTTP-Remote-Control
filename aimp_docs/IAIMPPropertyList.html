IAIMPPropertyList
Описание
IAIMPPropertyList представляет собой низкоуровневый интерфейс для доступа к дополнительным свойствам объекта, к которому он привязан.
Методы
BeginUpdate
Метод блокирует рассылку уведомлений об изменениях до вызова EndUpdate.
Рекомендуется к использованию в случае изменения нескольких свойств за один раз.
EndUpdate
Метод снимает блокировку с рассылки уведомлений об изменениях, что была поставлена вызовом метода BeginUpdate.
Reset
Сброс значений свойств списка в значение по умолчанию.

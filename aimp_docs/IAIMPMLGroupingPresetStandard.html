IAIMPMLGroupingPresetStandard
Описание
Интерфейс описывает шаблон, использующий стандартный провайдер данных для дерева группировки, и его настройки.
Предок
IAIMPMLGroupingPresetStandard наследуется от IAIMPMLGroupingPreset
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPML_GROUPINGPRESETSTD_PROPID_FIELDS
Список полей.
Каждая запись должна поддерживать минимум два интерфейса - IAIMPString (имя поля) и IAIMPMLSortDirection (направление сортировки).
ТипIAIMPObjectList
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPObjectList, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPObjectList>)
Методы
GetFilter
Возвращает ссылку на используемый фильтр.
outFilterIAIMPMLDataFilterGroup Изменения, сделанные в этом интерфейсе, автоматически применяются к шаблону.
Для множественных изменений в фильтре используйте BeginUpdate / EndUpdate.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

Использование GDI+
Если плагин использует в своем коде GDI+ API - инициализация и финализация библиотеки GDI+ (вызовы методов GdiplusStartup и GdiplusShutdown) должны вызываться строго в <IAIMPPlugin>.Initialize и <IAIMPPlugin>.Finalize согласно требованиям Microsoft (см. секцию "Remark").

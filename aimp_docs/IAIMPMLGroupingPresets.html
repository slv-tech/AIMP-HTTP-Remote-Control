IAIMPMLGroupingPresets
Описание
Предоставляет доступ к шаблонам дерева группировки конкретного хранилища данных.
Методы
Add
Добавляет шаблон с пользовательским провайдером данных, шаблон будет считаться Built-in, пользователь не сможет его изменить/удалить.
inIDIAIMPString ID шаблона, должен быть уникален среди всех плагинов.
inNameIAIMPString Имя шаблона.
Укажите nil/null, если используется автоматическая локализация по ID шаблона
inReservedLongWord Для будущего использования, должно быть равно 0.
inProviderIAIMPMLGroupingTreeDataProvider Провайдер данных для шаблона
outPresetIAIMPMLGroupingPreset Экземпляр созданного шаблона
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Add2
Добавляет шаблон со стандартным провайдером данных.
inIDIAIMPString ID шаблона, должен быть уникален среди всех плагинов.
inNameIAIMPString Имя шаблона.
Укажите nil/null, если используется автоматическая локализация по ID шаблона
inReservedLongWord Для будущего использования, должно быть равно 0.
inFieldNamesIAIMPObjectListСписок полей.
Каждая запись должна поддерживать минимум два интерфейса - IAIMPString (имя поля) и IAIMPMLSortDirection (направление сортировки).
outPresetIAIMPMLGroupingPresetStandard Экземпляр созданного шаблона
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Add3
Упрощенная версия метода Add2 - добавляет шаблон со стандартным провайдером данных на основе одного поля
inIDIAIMPString ID шаблона, должен быть уникален среди всех плагинов.
inNameIAIMPString Имя шаблона.
Укажите nil/null, если используется автоматическая локализация по ID шаблона
inReservedLongWord Для будущего использования, должно быть равно 0.
inFieldNameIAIMPString Имя поля в виде IAIMPString
outPresetIAIMPMLGroupingPresetStandard Экземпляр созданного шаблона
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
BeginUpdate
Метод блокирует рассылку уведомлений об изменениях до вызова EndUpdate.
Рекомендуется к использованию в случае изменения нескольких свойств за один раз.
EndUpdate
Метод снимает блокировку с рассылки уведомлений об изменениях, что была поставлена вызовом метода BeginUpdate.

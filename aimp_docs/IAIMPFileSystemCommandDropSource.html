IAIMPFileSystemCommandDropSource
Описание
Команда создает экземпляр IAIMPStream для указанного FileName для операции drag-n-drop.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
CreateStream
inFileNameIAIMPString Используется в качестве целевого имени для файла (без пути). При необходимости, вы можете изменить его.
outStreamIAIMPStream
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой

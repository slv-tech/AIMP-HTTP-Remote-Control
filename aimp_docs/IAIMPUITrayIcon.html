IAIMPUITrayIcon
Описание
IAIMPUITrayIcon позволяет плагину создать собственную иконку в области уведомлений.
Предок
IAIMPPropertyList -> IAIMPUITrayIcon
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_TRAYICON_ID
Идентификатор иконки, используется Windows для сопоставления настроек в области уведомления.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_TRAYICON_HINT
Подсказка, что отображается при наведении указателя мыши
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_TRAYICON_ICON
Используйте nil/null, чтобы использовать иконку приложения (значение по умолчанию)
ТипIAIMPImage, IAIMPImageContainer, IAIMPStream
ЧтениеНедоступно
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, ..)
AIMPUI_TRAYICON_VISIBLE
0 - невидима (по умолчанию).
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
Методы
Notify
Показывает сообщение в области уведомлений.
Клик по иконке вызовет IAIMPUIChangeEvents.
inTitleIAIMPStringЗаголовок
inMessageIAIMPStringСообщение
inFlagsLongWord Один из следующих флагов:
  • AIMPUI_TRAYICON_NOTIFY_FLAGS_INFO
  • AIMPUI_TRAYICON_NOTIFY_FLAGS_ERROR
  • AIMPUI_TRAYICON_NOTIFY_FLAGS_WARNING
    • inEventsIUnknown Клик по уведомлению вызовет IAIMPUIChangeEvents, если он реализуется в Events
    outResultHRESULTS_OK - Метод успешно завершился
    E_INVALIDARG - Один из параметров задан неверно
    E_NOTIMPL - Метод не реализован
    E_UNEXPECTED - Произошел непредвиденный сбой
    E_ACCESSDENIED - Метод был вызван не из главного потока
    Создание экземпляра
    Экземпляр создается с помощью <IAIMPServiceUI>.CreateObject.
    Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

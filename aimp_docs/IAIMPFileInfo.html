IAIMPFileInfo
Описание
Интерфейс IAIMPFileInfo представляет собой хранилище расширенной информации о файле.

Реализация интерфейса должна поддерживать асинхронную работу.
Предок
IAIMPFileInfo наследуется от IAIMPPropertyList.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMP_FILEINFO_PROPID_ML_MARK
Пользовательская оценка трека, хранимая в БД фонотеки.
Может принимать значения от 0 до 5.

Чтобы изменить значение, используйте метод указанный в поле Write из таблице ниже.
Обратите внимание, что метод может вернуть ошибку E_FAIL, если аудио файл не может быть импортирован в БД фонотеки.

AIMP_FILEINFO_PROPID_ML_LABELS
Пользовательские метки для аудио файла, что хранятся в БД фонотеки. Каждая метка отделяется от другой ";".

Чтобы изменить значение, используйте метод указанный в поле Write из таблице ниже.
Обратите внимание, что метод может вернуть ошибку E_FAIL, если аудио файл не может быть импортирован в БД фонотеки.

IAIMPUITreeListInplaceEditingEvents
Описание
События IAIMPUITreeListInplaceEditingEvents возникают при редактировании содержимого ячейки (если эта возможность включена)

Методы
OnEditing
Событие возникает при попытке запустить редактирование
inSenderIAIMPUITreeList
inNodeIAIMPUITreeListNode
inColumnIndexInteger Абсолютный порядковый номер колонки, ячейку которой пытается отредактировать пользователь.
varAllowLongBool Выставите переменную в True, если редактирование этой ячейки допустимо
OnEdited
Событие возникает при применении введенного пользователем значения
inSenderIAIMPUITreeList
inNodeIAIMPUITreeListNode
inColumnIndexInteger Абсолютный порядковый номер колонки, ячейку которой пытается отредактировать пользователь.
varValueIAIMPString Введенное пользователем значение.

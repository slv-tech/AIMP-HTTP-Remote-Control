IAIMPMLAlbumArtProvider2
Описание
Этот интерфейс реализует провайдер изображений для групп и миниатюр фонотеки, что реализуется на стороне плагина. В отличие от стандартного механизма предоставления обложек альбомов - сервиса IAIMPServiceAlbumArt - этот провайдер работает с уникальными для конкретной фонотеки данными.

Интерфейс должен быть реализован на том же уровне, что и IAIMPMLExtensionDataStorage.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
Get
inFieldsIAIMPObjectList Список имен полей, представленных в виде IAIMPString, у которых свойство AIMPML_FIELD_PROPID_FLAGS содержит AIMPML_FIELDFLAG_REQUIRED флаг.
inValuesPOleVariant Указатель на первый элемент массива OleVariant-ов. Размер массива равен значению Fields.GetCount.
Содержит значения полей первого файла из группы, для которой запрашивается обложка альбома.
inRequestIAIMPAlbumArtRequestПользовательские настройки поиска обложек альбомов, см. IAIMPAlbumArtRequest.

Внимание! Перед выполнение запроса провайдер обязан проверить текущие настройки, и, в случае, если они не удовлетворяют его потребностям, завершить работу с ошибкой E_FAIL.

Рассмотрим взаимодействие с IAIMPAlbumArtRequest на примере провайдера, скачивающего картинку из интернета:

function Get(FileInfo: IAIMPFileInfo; Request: IAIMPAlbumArtRequest; out Image: IAIMPImageContainer): HRESULT; stdcall;
var
  ALink: IAIMPString;
begin
  Result := E_FAIL;
  // Проверяем, задана ли у файла ссылка?
  if PropListGetStr(FileInfo, AIMP_FILEINFO_PROPID_URL, ALink) then
  begin 
    // Это ссылка на картинку?
    if IsImageURL(ALink) then 
    begin
      // Она уже есть в кэше? 
      if Succeeded(Request.CacheGet(ALink, Image)) then
        Exit(S_OK);
        
      // Скачиваем картинку
      // Функция Download самостоятельно проверит и учтёт значения опций 
      // AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET и AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET_MAX_FILE_SIZE
      if Succeeded(Request.Download(ALink, Image)) then
      begin
        // Оптимизируем и обновляем кэш 
        Request.CachePut(ALink, Image);
        Result := S_OK;
      end;
    end;
  end;
end;
outImageIAIMPImageContainer Картинка для группы
outResultHRESULT S_OK - если провайдер успешно завершил работу
E_FAIL - возникла ошибка или провайдер ничего не нашел
E_NOTIMPL - провайдер не функционирует, вызов должен быть передан стандартному провайдеру обложек

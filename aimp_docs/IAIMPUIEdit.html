IAIMPUIEdit
Описание
IAIMPUIEdit представляет собой однострочный текстовый редактор
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIBaseEdit -> IAIMPUIBaseButtonnedEdit -> IAIMPUIEdit
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_EDIT_PROPID_PASSWORDCHAR
Код символа, который будет отображаться вместо введенных пользователем символов.

Если указать 0 - введенный текст будет отображаться нормально.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_EDIT_PROPID_TEXTHINT
Текст, который будет отображаться при пустом значении поля ввода.
Только для Windows Vista и более новых ОС.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
События
IAIMPUIChangeEvents вызывается при изменении текста
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

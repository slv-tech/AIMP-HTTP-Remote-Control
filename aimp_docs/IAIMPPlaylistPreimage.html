IAIMPPlaylistPreimage
Описание
Интерфейс представляет собой базовый интерфейс для всех прообразов.

Прообраз - источник данных для смарт-плейлистов, с ручной или автоматической синхронизаций данных. Принцип взаимодействия прообраза и плейлиста следующий: когда в прообразе меняются данные или настройки, он должен уведомить об этом IAIMPPlaylistPreimageListener, который прикреплен к определенному плейлисту. Плейлист, в свою очередь, запросит новые данные из прообраза только тогда, когда они понадобятся - это может произойти как сразу, так и спустя какое-то время.
Предок
IAIMPPlaylistPreimage наследуется от IAIMPPropertyList

Значение отличное от 0 - в прообразе включена автоматическая синхронизация данных с источником (не все прообразы поддерживают это свойство).
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_PLAYLISTPREIMAGE_PROPID_AUTOSYNC_ON_STARTUP
Значение отличное от 0 - в прообразе включена автоматическая синхронизация данных с источником при запуске программы (не все прообразы поддерживают это свойство).
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_PLAYLISTPREIMAGE_PROPID_HASDIALOG
Значение отличное от 0, если в прообразе реализован ExecuteDialog метод.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
AIMP_PLAYLISTPREIMAGE_PROPID_SORTTEMPLATE
Шаблон для сортировки значений в плейлисте после добавления в него файлов из прообраза
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)

AIMP_RA_PROPERTY_VERSION
AIMP_RA_PROPERTY_VERSION
Версия плеера. Только чтение
GETParamLPARAMне используется
GETResultLRESULT HiWord(Result) - ID версии
LoWord(Result) - Номер сборки

ID версии в формат XYY.
X - мажорная версия, не ограничена по количеству разрядов
YY - минорная версия, ограничена двумя разрядами

Примеры:
355 -> 3.55
1005 -> 10.05
001 -> 0.01


Получение значения:
    Window = FindWindow(AIMPRemoteAccessClass, null);
    ...
    Result = SendMessage(Window, WM_AIMP_PROPERTY, ID or AIMP_RA_PROPVALUE_GET, Param);

AIMP_MSG_CMD_QFI
AIMP_MSG_CMD_QFI
Показывает окно "Quick File Information" для указанного файла
inParam1Integer не используется
inParam2Pointer Указатель на структуру TAIMPQuickFileInfoParams
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
TAIMPQuickFileInfoParams
incbSizeInteger Размер структуры
inCmdShowInteger Команда на показ / скрытие окна.
Может принимать одно из следующих значений:
AIMP_QFI_SW_HIDE
AIMP_QFI_SW_SHOW
inAnimationTypeInteger Анимация при показе / скрытии окна.
Может принимать одно из следующих значений:
AIMP_QFI_ANIMATION_NONE
AIMP_QFI_ANIMATION_FADE
inAnimationTimeInteger Длительность анимации, в миллисекундах.
inDisplayTimeInteger Время отображения на экране, в миллисекундах.
Установите значение в 0, чтобы использовать время отображения по умолчанию.
inOpacityByte Непрозрачность, в процентах: 0..100%
inFileInfoIAIMPFileInfo Ссылка на информацию о файле, которую будет отображать окно

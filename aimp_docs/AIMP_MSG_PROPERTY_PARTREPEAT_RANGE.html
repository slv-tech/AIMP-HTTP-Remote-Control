AIMP_MSG_PROPERTY_PARTREPEAT_RANGE
AIMP_MSG_PROPERTY_PARTREPEAT_RANGE
Предоставляет доступ к диапазону повтора участка трека. Вы можете как получить текущий диапазон, так и модифицировать его.
inParam1Integer Направление доступа: чтение / запись (AIMP_MSG_PROPVALUE_GET / AIMP_MSG_PROPVALUE_SET)
inParam2Pointer Указатель на первый элемент массива из двух Single-элементов (32-ые значения с плавающей запятой).

1-ый элемент - точка начала повтора (в секундах), -1 если точка не задана.
2-ый элемент - точка окончания повтора (в секундах), -1 если точка не задана.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой

IAIMPUIFormEvents
Описание
IAIMPUIFormEvents позволяет обрабатывать события окна верхнего уровня (см. IAIMPUIForm)
Методы
OnActivated / OnDeactivated
События вызываются при изменении активности окна.

Только для окон верхнего уровня.
inSenderIUnknown Элемент управления, вызвавший событие
OnCreated / OnDestroyed
События вызываются при создании / убиении окна.
inSenderIUnknown Элемент управления, вызвавший событие
OnCloseQuery
Событие вызывается при попытке закрытия окна.

Только для окон верхнего уровня.
inSenderIUnknown Элемент управления, вызвавший событие
varCanCloseLongBool Плагин должен выставить флаг в False, чтобы запретить закрытие окна
OnLocalize
Событие вызывается при изменении языка приложения.

Обратите внимание на статью "автоматическая локализация".
inSenderIUnknown Элемент управления, вызвавший событие
OnShortCut
Событие вызывается при нажатии клавиш или их сочетаний в пределах этого окна, служит для ручной реализации поддержки локальных горячих клавиш.
inSenderIUnknown Элемент управления, вызвавший событие
inKeyWordВиртуальный код нажатой клавиши
inModifiersWord 0 или комбинация из следующих флагов:
  • AIMPUI_FLAGS_MOD_ALT
  • AIMPUI_FLAGS_MOD_CTRL
  • AIMPUI_FLAGS_MOD_SHIFT
varHandledLongBool Если сочетание является горячей клавишей - плагин должен выставить Handled в True
Создание экземпляра
Экземпляр создается плагином и передается в метод <IAIMPServiceUI>.CreateControl при создании элемента управления.

IAIMPServiceFileInfo
Описание
Сервис предоставляет возможность получения информации о виртуальном файле по его URI, а так же возможность получения расширенной информации о файле на основе его имени и/или потока.
Методы
GetFileInfoFromFileURI
inFileURIIAIMPString Имя виртуального файла
inFlagsLongWordКомбинация из следующих флагов:
  • AIMP_SERVICE_FILEINFO_FLAG_DONTUSEAUDIODECODERS - запретить получение информации с помощью IAIMPServiceAudioDecoders
    • inInfoIAIMPFileInfo
    outResultHRESULTS_OK - Метод успешно завершился
    E_INVALIDARG - Один из параметров задан неверно
    E_NOTIMPL - Метод не реализован
    E_UNEXPECTED - Произошел непредвиденный сбой
    GetFileInfoFromStream
    inStreamIAIMPStreamВнимание: Данный метод работает медленнее, чем GetFileInfoFromFileURI.
    inFlagsLongWordКомбинация из следующих флагов:
  • AIMP_SERVICE_FILEINFO_FLAG_DONTUSEAUDIODECODERS - запретить получение информации с помощью IAIMPServiceAudioDecoders
    • inInfoIAIMPFileInfo
    outResultHRESULTS_OK - Метод успешно завершился
    E_INVALIDARG - Один из параметров задан неверно
    E_NOTIMPL - Метод не реализован
    E_UNEXPECTED - Произошел непредвиденный сбой
    GetVirtualFile
    inFileURIIAIMPString Имя виртуального файла
    inFlagsLongWord Для будущего использования, должен быть равен нулю.
    outInfoIAIMPVirtualFile
    outResultHRESULT S_OK - метод завершился успешно,
    любое другое значение - файл не является виртуальным.
    Доступ к сервису
    <IAIMPCore>.QueryInterface(IID_IAIMPServiceFileInfo, ...);

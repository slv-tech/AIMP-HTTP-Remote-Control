IAIMPUIPageControlEvents
Описание
IAIMPUIPageControlEvents представляет собой интерфейс для обработки событий переключения вкладок IAIMPUIPageControl.
Методы
OnActivating
Событие вызывается перед переходом на страницу
inSenderIAIMPUIPageControl
inPageIAIMPUITabSheet Страница, которая будет активирована
varAllowLongBool Чтобы запретить переход на эту страницу, плагин должен выставить этот параметр в False.
OnActivated
Событие вызывается после перехода на страницу
inSenderIAIMPUIPageControl
inPageIAIMPUITabSheet Активная страница
Создание экземпляра
Экземпляр создается плагином и передается в метод <IAIMPServiceUI>.CreateControl при создании элемента управления.

IAIMPUILabel
Описание
IAIMPUILabel представляет собой элемент для вывода текста или ссылок.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUILabel
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_LABEL_PROPID_AUTOSIZE
Автоматическая подгонка размеров элемента управления под размеры его содержимого. 0 - выключено (значение по умолчанию).
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_LABEL_PROPID_LINE
Показывать ли линию разделитель справа от текста. 0 - линия скрыта.

Свойство AIMPUI_LABEL_PROPID_AUTOSIZE должно быть выставлено в 0.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_LABEL_PROPID_TEXT
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_LABEL_PROPID_TEXTALIGN
Выравнивание текста по горизонтали, может принимать одно из значений:
  • AIMPUI_ALIGN_NEAR - (left) (по умолчанию)
  • AIMPUI_ALIGN_CENTER
  • AIMPUI_ALIGN_FAR - (right)
    • ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_LABEL_PROPID_TEXTALIGNVERT
    Выравнивание текста по вертикали, может принимать одно из значений:
  • AIMPUI_ALIGN_NEAR - (top)
  • AIMPUI_ALIGN_CENTER - (по умолчанию)
  • AIMPUI_ALIGN_FAR - (bottom)
    • ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32

    AIMPUI_LABEL_PROPID_TEXTCOLOR
    Цвет текста в формате RRGGBB
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_LABEL_PROPID_TEXTSTYLE
    Начертание шрифта.
    0 или комбинация из следующих флагов:

    + AIMPUI_FLAGS_FONT_BOLD
    + AIMPUI_FLAGS_FONT_ITALIC
    + AIMPUI_FLAGS_FONT_STRIKEOUT
    + AIMPUI_FLAGS_FONT_UNDERLINE

    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_LABEL_PROPID_TRANSPARENT
    Свойство показывает, нужно ли рисовать фон у элемента.
    0 - рисовать фон нужно.
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_LABEL_PROPID_URL
    Ссылка, открывается в браузере, когда пользователь кликает по текстовой метке.
    ТипIAIMPString
    ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
    ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
    AIMPUI_LABEL_PROPID_WORDWRAP
    Свойство показывает, можно ли переносить текст по словам, если он не влезает.
    0 - перенос по словам выключен.
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    Создание экземпляра
    Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
    Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

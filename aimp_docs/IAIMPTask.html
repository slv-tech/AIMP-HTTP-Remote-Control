IAIMPTask
Описание
Должен реализовываться плагином и является интерфейсом доступа к объекту, выполняющему задачу в асинхронном режиме.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
Execute
inOwnerIAIMPTaskOwner Интерфейс позволяет узнать об отмене задачи
Рекомендации по реализации
Стоит по подробнее остановиться на реализации метода Execute. Основной особенность является то, что плагин обязан как можно чаще вызывать функцию обратного вызова и адекватно реагировать на возвращаемый ею результат.

Например:
procedure <IAIMPTask>.Execute(Owner: IAIMPTaskOwner);
begin
  //задача отменена?
  if Owner.IsCanceled then 
    // отменяем выполнение задачи  
  
  // делаем что-нибудь
   
  //задача отменена?
  if Owner.IsCanceled then 
    // отменяем выполнение задачи
      
  // делаем еще что-нибудь 
end;

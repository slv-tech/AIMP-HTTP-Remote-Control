Введение
AIMP имеет два типа API - Local и Remote.

Local API предоставляется непосредственно плагинам, что представлены в виде динамически подключаемых библиотек (DLL) и располагаются в подпапке Plugins папки плеера. Этот тип API представляет максимальные возможности взаимодействия с модулями программы.

Remote API представляет собой урезанную версию Local API, и предоставляется для сторонних, независимо работающих программ. Данный тип API не должен использоваться в плагинах.

IAIMPExtensionPlayerHook
Описание
Это расширение сервиса IAIMPServicePlayer позволяет плагину проверить и, при необходимости, подкорректировать внешний URL. Это может быть полезно, если удаленный сервис требует дополнительные параметры для предоставление ссылки на аудиопоток, которые нельзя хранить в файле плейлиста (например, авторизационные данные).

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
OnCheckURL
Событие вызывается при попытке проиграть удаленный URL в плеере.
inURLIAIMPString
varHandledLongBool Если плагин обработал вызов - он должен выставить Handled в True.
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

IAIMPUIPopupMenuEvents
Описание
IAIMPUIPopupMenuEvents представляет собой интерфейс для обработки события вызова контекстного меню для элемента.

Обратите внимание, что если вы назначили IAIMPUIPopupMenu элементу управления - обрабатывать это событие не нужно.
Методы
OnContextPopup
inSenderIUnknown Элемент управления, вызвавший событие
inX/YInteger Координаты мыши в системе координат элемента управления.
(0, 0) - верхний левый угол.
outResultLongBool Плагин должен вернуть True, если он обработал вызов и вызывать контекстное меню по умолчанию не нужно. В противном случае - False.
Создание экземпляра
Экземпляр создается плагином и передается в метод <IAIMPServiceUI>.CreateControl при создании элемента управления.

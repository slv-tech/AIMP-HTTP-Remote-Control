IAIMPUIRadioBox
Описание
Элемент управления представляет из себя круглое поле с возможностью установки точки в нем. Разместите несколько таких элементов управления, чтобы предоставить пользователю выбор только одного из них.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUICheckBox -> IAIMPUIRadioBox
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

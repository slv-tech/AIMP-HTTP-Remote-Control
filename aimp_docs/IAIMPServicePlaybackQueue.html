IAIMPServicePlaybackQueue
Описание
Сервис предоставляет доступ к внутренней очереди воспроизведения (не стоит путать с IAIMPPlaylistQueue).
Внутренняя очередь воспроизведения по умолчанию строится на основе содержимого плейлиста и значения свойства "Shuffle", однако может быть переопределена расширением IAIMPServicePlaybackQueue.
Методы
GetNextTrack
Функция ищет и возвращает следующий для воспроизведения трек, основываясь на текущем воспроизводимом файле.
outItemIAIMPPlaybackQueueItem
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetPrevTrack
Функция ищет и возвращает предыдущий трек, основываясь на текущем воспроизводимом файле.
outItemIAIMPPlaybackQueueItem
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServicePlaybackQueue, ...);

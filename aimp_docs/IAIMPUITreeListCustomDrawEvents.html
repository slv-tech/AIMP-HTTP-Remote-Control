IAIMPUITreeListCustomDrawEvents
Описание
IAIMPUITreeListCustomDrawEvents позволяет переопределить отрисовку узла или ячейки.

Методы
OnCustomDrawNode
Событие возникает при отрисовке фона строки узла
inSenderIAIMPUITreeList
inDCHDC Дескриптор контекста
inRTRect Координаты строки узла для заданного контекста
inNodeIAIMPUITreeListNode Узел, который в данный момент рисуется
varHandledLongBool Выставите переменную в True, чтобы подавить отрисовку по умолчанию.
Стоит отметить, что в таком случае событие OnCustomDrawNodeCell уже не вызовется.
OnCustomDrawNodeCell
Событие возникает при отрисовке ячейки строки узла
inSenderIAIMPUITreeList
inDCHDC Дескриптор контекста
inRTRect Координаты ячейки для заданного контекста
inNodeIAIMPUITreeListNode Узел, который в данный момент рисуется
inColumnIAIMPUITreeListColumn Колонка, ячейка которой, в данный момент рисуется
varHandledLongBool Выставите переменную в True, чтобы подавить отрисовку по умолчанию.
OnGetNodeBackground
Событие позволяет переопределить цвет фона строки узла
inSenderIAIMPUITreeList
inNodeIAIMPUITreeListNode Узел, который в данный момент рисуется
varColorLongWord Цвет фона в формате RRGGBB

TAIMPServiceFindTagsOnlineAlbumInfoReceiveProc
Описание
TAIMPServiceFindTagsOnlineAlbumInfoReceiveProc - функция обратного вызова для передачи информации о найденном плагином альбоме.
Плагин может вызвать функцию несколько раз, в зависимости от того, сколько альбомов он нашел по запросу.

inAlbumInfoIAIMPFileInfo Информация о найденном альбоме
inDataPointer Пользовательский параметр, передается плагину при вместе с функцией обратного вызова.
Плагин должен передать его при вызове функции обратного вызова.

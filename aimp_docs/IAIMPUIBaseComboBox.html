IAIMPUIBaseComboBox
Описание
IAIMPUIBaseComboBox представляет собой базовый интерфейс для полей ввода с выпадающим списком.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIBaseEdit -> IAIMPUIBaseButtonnedEdit -> IAIMPUIBaseComboBox

IAIMPMenuItem
Описание
IAIMPMenuItem описывает пункт меню.
Предок
IAIMPMenuItem наследуется от IAIMPPropertyList.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMP_MENUITEM_PROPID_CUSTOM
Пользовательский параметр, позволяет прикрепить к экземпляру интерфейса пользовательский набор данных
ТипIUnknown
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IUnknown, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IUnknown>)
AIMP_MENUITEM_PROPID_ACTION
Свойство позволяет связать пункт меню с уже созданной командой.
При этом пункт меню наследует из команды значения свойств AIMP_MENUITEM_PROPID_NAME, AIMP_MENUITEM_PROPID_ENABLED, AIMP_MENUITEM_PROPID_EVENT.
ТипIAIMPAction
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPAction, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(.., IID_IAIMPAction, ..)
AIMP_MENUITEM_PROPID_ID
Уникальный идентификатор пункта меню, должен однозначно идентифицировать пункт меню среди всех плагинов
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_MENUITEM_PROPID_CAPTION
Имя пункта меню.

Значение актуально только в том случае, если значение свойства AIMP_MENUITEM_PROPID_STYLE отлично от AIMP_MENUITEM_STYLE_NORMAL.

ЗаписьIAIMPPropertyList.SetValueAsInt32

AIMP_MENUITEM_STYLE_NORMAL, AIMP_MENUITEM_STYLE_CHECKBOX или AIMP_MENUITEM_STYLE_RADIOBOX

Методы
DeleteChildren
Метод вызывает IAIMPCore.UnregisterExtension для всех подпунктом меню.

Смотрите также: Процесс удаления пункта меню
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Создание экземпляра
Экземпляр создается с помощью <IAIMPCore>.CreateObject(IID_IAIMPMenuItem, ...) при инициализации плагина.

После заполнения необходимых полей экземпляра, плагин обязан зарегистрировать пункт меню в менеджере с помощью <IAIMPCore>.RegisterExtension(IID_IAIMPServiceMenuManager, <IAIMPMenuItem>).
Удаление пункта меню
Все пункты меню удаляются автоматически перед финализацией плагина. Однако, если вам необходимо удалить пункт меню в процессе работы программы - воспользуйтесь методом IAIMPCore.UnregisterExtension. Этот метод снимает пункт меню и его подпункты с регистрации, а так же сбрасывает у них свойство AIMP_MENUITEM_PROPID_PARENT в значение по умолчанию (nil / null). Для повторного использования удаленного пункта меню, необходимо заново про инициализировать его и зарегистрировать в программе.

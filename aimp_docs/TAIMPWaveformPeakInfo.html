TAIMPWaveformPeakInfo
Описание
Структура содержит информацию о точке для waveform-навигатора.

TAIMPWaveformPeakInfo
inMaxNegativeWord Максимальное по модулю значение среди отрицательных в блоке данных, который описывает точка.
inMaxPositiveWord Максимальное по модулю значение среди положительных в блоке данных, который описывает точка.

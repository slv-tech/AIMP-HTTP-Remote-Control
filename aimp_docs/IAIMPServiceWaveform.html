IAIMPServiceWaveform
Описание
Сервис предоставляет возможность зарегистрировать собственный провайдер данных для waveform-навигатора в плеере.
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceWaveform, ...);
Резюме
Объявление:apiPlayer.pas / apiPlayer.h
Асинхронная работа:Не поддерживается
Расширения:IAIMPExtensionWaveformProvider
Приложения:Player

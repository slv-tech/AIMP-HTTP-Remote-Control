IAIMPUIProgressDialog
Описание
IAIMPUIProgressDialog предоставляет собой диалог прогресса выполнения какой-либо задачи. Это единственный диалог, который поддерживает асинхронную работу - вызовы из второстепенных потоков.

Свойства
Свойства должны быть выставлены до того, как диалог будет показан.
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_PROGRESSDLG_PROPID_CAPTION
Заголовок диалога
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_PROGRESSDLG_PROPID_MESSAGE
Сообщение по умолчанию, что будет отображаться над полосой прогресса.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_PROGRESSDLG_PROPID_SHOW_PROGRESS_ON_TASKBAR
Свойство позволяет включить показ прогресса на панели задач
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32

Методы
Started
Метод показывает диалог
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Finished
Метод закрывает диалог
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Progress
Метод обновляет прогресс в диалоге.
Метод должен вызываться строго после вызова Started и до вызова Finished.
inPositionInt64
inTotalInt64
inTextIAIMPString Текст, что будет отображаться над полосой прогресса. Может быть равен nil / null, в этом случае в качестве текста будет использоваться значение свойства AIMPUI_PROGRESSDLG_PROPID_MESSAGE
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Доступ к интерфейсу
<IAIMPServiceUI>.CreateObject(.., <IAIMPUIProgressDialogEvents>, IID_IAIMPUIProgressDialog, <IAIMPUIProgressDialog>)

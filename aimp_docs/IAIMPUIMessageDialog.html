IAIMPUIMessageDialog
Методы
Execute
Все параметры такие же, как для стандартного MessageBox-а.
inOwnerWndHWND Дескриптор окна-владельца диалога.
Если указать 0 - будет использоваться дескриптор главного окна.
inCaptionIAIMPString
inTextIAIMPString
inFlagsLongWord
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

Дополнительно, обратите внимание на секцию "Return value" в статье про системный MessageBox.
Доступ к интерфейсу
<IAIMPServiceUI>.QueryInterface(IID_IAIMPUIMessageDialog, <IAIMPUIMessageDialog>)

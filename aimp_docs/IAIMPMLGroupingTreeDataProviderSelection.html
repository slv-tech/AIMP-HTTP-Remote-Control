IAIMPMLGroupingTreeDataProviderSelection
Описание
Интерфейс предоставляет доступ к текущей выборке данных от IAIMPMLGroupingTreeDataProvider.

Реализация интерфейса должна поддерживать асинхронную работу.
Методы
GetDisplayValue
Возвращает текстовое представление данных для отображения пользователю (может отсутствовать)
outSIAIMPString
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.

В случае ошибки, движок программы построит отображаемое значение сам, на основе типа поля.
GetFlags
outResultLongWord 0 или комбинация флагов:
  • AIMPML_GROUPINGTREENODE_FLAG_HASCHILDREN - текущий узел имеет подуровни
  • AIMPML_GROUPINGTREENODE_FLAG_STANDALONE - см.IAIMPMLGroupingTreeSelection
    • GetImageIndex
    outImageIndexInteger Номер картинки в дереве группировки, что будет отображаться на уровне с этим полем:
    • AIMPML_FIELDIMAGE_FOLDER
    • AIMPML_FIELDIMAGE_ARTIST
    • AIMPML_FIELDIMAGE_DISK
    • AIMPML_FIELDIMAGE_NOTE
    • AIMPML_FIELDIMAGE_STAR
    • AIMPML_FIELDIMAGE_CALENDAR
    • AIMPML_FIELDIMAGE_LABEL
    outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
    GetValue
    outFieldNameIAIMPString
    outValueOleVariant
    outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
    NextRecord
    Переход к следующему узлу на текущем уровне. Если текущий узел последний - функция должна вернуть False.
    outResultLongBool

IAIMPExtensionPlaybackQueue2
Описание
Расширяет возможности IAIMPExtensionPlaybackQueue.

Методы
GetInfo
Возвращает информация о размерности очереди воспроизведения и позиции текущей записи в ней.
inCurrentIUnknown Текущая запись, IAIMPPlaylistItem.
outPositionInteger Позиция текущей записи в очереди воспроизведения (начиная с 1). 0 - если позиция неизвестна.
outSizeInteger Размер очереди воспроизведения. 0 - если размер неизвестен.
outResultLongBool

Уведомление об изменениях
При изменении порядка записей в очереди со стороны плагина, он должен уведомить об этом плеер посредством вызова IAIMPServicePlaybackQueue2.NotifyChanged.

Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

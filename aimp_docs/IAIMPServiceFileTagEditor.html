IAIMPServiceFileTagEditor
Методы
EditFile
Этот метод позволяет редактировать все поддерживаемые теги в указанном файле.
Метод проверяет формат файла перед редактированием, а также считывает информацию о качестве и длительности аудиопотока
inSourceIUnknownВ качестве источника может выступать имя файла (IAIMPString) или поток (IAIMPStream)
inIIDTGUID IID_IAIMPFileTagEditor
outObjectUntypedIAIMPFileTagEditor
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_FAIL - формат файла не поддерживается редактором тегов
EditTag
В отличие от EditFile, этот метод пытается считать указанный в TagID тип тега из файла. Метод не проверяет формат на поддерживаемый и не предоставляет информации о аудиопотоке.

Внимание! Текущая реализация не поддерживает запись тега в файл.
inSourceIUnknownВ качестве источника может выступать имя файла (IAIMPString) или поток (IAIMPStream)
inTagIDInteger AIMP_FILETAG_ID_APEv2
AIMP_FILETAG_ID_ID3v1
AIMP_FILETAG_ID_ID3v2
AIMP_FILETAG_ID_VORBIS
AIMP_FILETAG_ID_M4A
AIMP_FILETAG_ID_WMA
inIIDTGUID IID_IAIMPFileInfo
outObjectUntypedIAIMPFileInfo
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_FAIL - тег не найден в указанном файле
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceFileTagEditor, ...);

IAIMPUIValidationLabel
Описание
IAIMPUIValidationLabel, в отличие от IAIMPUILabel, отображает статус-иконку слева от текста.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUILabel -> IAIMPUIValidationLabel
Свойства
AIMPUI_VALIDATIONLABEL_PROPID_GLYPH
Индекс иконки, может принимать одно из значений:
  • AIMPUI_VALIDATIONLABEL_GLYPH_OK - "все хорошо"
  • AIMPUI_VALIDATIONLABEL_GLYPH_WARNING - "предупреждение"
  • AIMPUI_VALIDATIONLABEL_GLYPH_ERROR - "ошибка"
  • AIMPUI_VALIDATIONLABEL_GLYPH_CRITICAL_WARNING - "важное предупреждение"
    • ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    Создание экземпляра
    Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
    Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

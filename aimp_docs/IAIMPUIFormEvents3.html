IAIMPUIFormEvents3
Описание
IAIMPUIFormEvents3 позволяет обрабатывать события окна верхнего уровня (см. IAIMPUIForm)
Методы
OnStyleChanged
Событие вызывается при переключении между дневным / ночным стилями или изменении цветовой схемы.

Узнать текущий стиль можно с помощью свойства AIMPUI_FORM_PROPID_STYLE

Только для окон верхнего уровня.
inSenderIUnknown Элемент управления, вызвавший событие
inStyleInteger Может принимать значения AIMPUI_STYLE_LIGHT / AIMPUI_STYLE_DARK.
Создание экземпляра
Экземпляр создается плагином и передается в метод <IAIMPServiceUI>.CreateControl при создании элемента управления.

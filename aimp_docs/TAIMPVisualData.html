TAIMPVisualData
TAIMPVisualData
Содержит в себе данные для отрисовки кадра визуализации.
Обратите внимание: данные рассчитываются без учета громкости, баланса и антиклиппинга.
inPeaks(2)Single Массив из двух элементов типа Single.
Уровень сигнала в левом и правом каналах [0.0 .. 1.0 .. бесконечность], где 1.0 равно 0 дБ.
inSpectrum(3)TAIMPVisualDataSpectrum Массив из трех элементов типа TAIMPVisualDataSpectrum.
0 - спектр левого канала
1 - спектр правого канала
2 - спектр суммы левого и правого каналов

Данные заполняются только в том случае, если расширение вернуло в GetFlags флаг AIMP_VISUAL_FLAGS_RQD_DATA_SPECTRUM.
inWaveForm(2)TAIMPVisualDataWaveform Массив из двух элементов типа TAIMPVisualDataWaveform.
Исходные данные сигнала в виде "волны" для левого и правого каналов.

Данные заполняются только в том случае, если расширение вернуло в GetFlags флаг AIMP_VISUAL_FLAGS_RQD_DATA_WAVEFORM.

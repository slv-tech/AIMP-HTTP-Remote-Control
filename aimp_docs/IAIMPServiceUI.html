IAIMPServiceUI
Описание
GUI - фреймворк для построения интерфейсов окон плагина в стилистике используемой версии AIMP.

Сервис предоставляет доступ к основным функциям создания окон, визуальных и невизуальных элементов.
Методы
CreateForm
Метод создает окно. Окно является главным элементом - на нем располагаются все остальные элементы - как визуальные, так и невизуальные.
inOwnerWindowHWND Дескриптор окна-владельца. Созданное окно будет держаться всегда поверх своего владельца.
inFlagsLongWord 0 или один из флагов:
AIMPUI_SERVICE_CREATEFORM_FLAGS_CHILD - при указании этого флага, созданное окно будет вести себя как дочерний элемент управления окна-владельца.
inNameIAIMPStringИмя элемента, может содержать только латинские буквы и цифры. Используется для автоматической локализации
inEventsHandlerIUnknown Обработчик событий, реализуется на стороне плагина. Может быть равен nil / null.
outFormIAIMPUIForm Интерфейс только что созданного окна
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
CreateControl
Метод создает визуальный элемент управления привязанный к определенному окну.
inOwnerIAIMPUIForm Окно-владелец. Все дочерние элементы убиваются автоматически при убиении окна.
inParentIAIMPUIWinControl Элемент управления или окно, непосредственно на котором будет лежать создаваемый элемент управления.
Parent должен принадлежать тому же окну, что и создаваемый элемент.
inNameIAIMPStringИмя элемента, может содержать только латинские буквы и цифры. Используется для автоматической локализации
inIIDTGUID Идентификатор создаваемого элемента управления
inEventsHandlerIUnknown Обработчик событий, реализуется на стороне плагина. Может быть равен nil / null.
outControl Интерфейс только что созданного элемента управления
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
CreateObject
Метод создает невизуальный элемент управления привязанный с опциональной привязкой к окну.
inOwnerIAIMPUIForm Окно-владелец. Все дочерние элементы убиваются автоматически при убиении окна.
Укажите nil / null, если объект не нужно привязывать к окну. В этом случае, объект будет жить до тех пор, пока на его интерфейс есть ссылки.
inEventsHandlerIUnknown Обработчик событий, реализуется на стороне плагина. Может быть равен nil / null.
inIIDTGUID Идентификатор создаваемого элемента управления
outObj Интерфейс только что созданного элемента управления
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceUI, ...);

IAIMPUIForm
Описание
IAIMPUIForm представляет из себя окно верхнего уровня, которое содержит все остальные элементы управления.

Обратите внимание, что в отличии от всех других элементов, здесь не действует ARC. Форма убивается при ее закрытии пользователем, или после ручного вызова метода Release. Убедитесь, что после этих событий все ссылки на форму (если таковые имеются) очищены, в противном случае это может привести к вылету плагина и самого приложения.

Форма имеет два режима отображения: как самостоятельное окно верхнего уровня, как дочернее окно другой формы. Режим отображения задается при создании формы.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIForm

AIMPUI_FORM_PROPID_BORDERSTYLE
Стиль рамки окна, принимает один из флагов:
+ AIMPUI_FLAGS_BORDERSTYLE_SIZEABLE - обычное окно
+ AIMPUI_FLAGS_BORDERSTYLE_SINGLE - обычное окно без возможности менять размер
+ AIMPUI_FLAGS_BORDERSTYLE_DIALOG - диалоговое окно без возможности менять размер, не имеет иконки и кнопок минимизации / максимизации окна
+ AIMPUI_FLAGS_BORDERSTYLE_TOOLWINDOW - окно в стиле панели инструментов без возможности менять размер
+ AIMPUI_FLAGS_BORDERSTYLE_TOOLWINDOWSIZABLE - окно в стиле панели инструментов с возможностью менять размер
+ AIMPUI_FLAGS_BORDERSTYLE_NONE - окно без рамки и заголовка, рекомендуется для встраиваемых форм

ShowModal
outResultInteger Результат вызова модального диалога, может принимать одно из значений:

+ AIMPUI_FLAGS_MODALRESULT_NONE
+ AIMPUI_FLAGS_MODALRESULT_OK
+ AIMPUI_FLAGS_MODALRESULT_CANCEL
+ AIMPUI_FLAGS_MODALRESULT_ABORT
+ AIMPUI_FLAGS_MODALRESULT_RETRY
+ AIMPUI_FLAGS_MODALRESULT_IGNORE
+ AIMPUI_FLAGS_MODALRESULT_YES
+ AIMPUI_FLAGS_MODALRESULT_NO
+ AIMPUI_FLAGS_MODALRESULT_CLOSE
+ AIMPUI_FLAGS_MODALRESULT_HELP
+ AIMPUI_FLAGS_MODALRESULT_TRYAGAIN
+ AIMPUI_FLAGS_MODALRESULT_CONTINUE


Какие из флагов будут использовать в форме - определяется настройками дочерних элементов, таких как "кнопка".

Флаг AIMPUI_FLAGS_MODALRESULT_CANCEL используется всегда при закрытии формы кликом по крестику в заголовке или горячей клавишей.

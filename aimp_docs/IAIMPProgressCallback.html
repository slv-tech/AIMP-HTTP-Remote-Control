IAIMPProgressCallback
Описание
Интерфейс предоставляет программе возможность показывать прогресс выполнения операции и отменить ее в случае необходимости.

Экземпляр интерфейса передается плагину при вызове какой-либо команды. Плагин, в свою очередь, должен: вызывать метод Progress данного интерфейса в процессе выполнения операции, проверять состояние флага Canceled после каждого вызова, в случае отмены операции - прервать ее.
Методы
Process
inProgressSingle Текущий прогресс выполнения операции, в процентах
varCanceledLongBool Вызывающая сторона может отменить текущий процесс, в этом случае флаг Canceled станет True.
Создание экземпляра
Экземпляр создается вызывающей стороной.

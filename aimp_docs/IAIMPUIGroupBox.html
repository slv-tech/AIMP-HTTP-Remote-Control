IAIMPUIGroupBox
Описание
IAIMPUIGroupBox позволяет создать контейнер-группу для других элементов управления.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIGroupBox
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_GROUPBOX_PROPID_AUTOSIZE
Автоматическая подгонка размеров элемента управления под размеры его содержимого. 0 - выключено (значение по умолчанию).
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_GROUPBOX_PROPID_BORDERS
Свойство указывает, у каких сторон нужно рисовать рамку.

0 или комбинация из флагов:
+ AIMPUI_FLAGS_BORDER_LEFT
+ AIMPUI_FLAGS_BORDER_TOP
+ AIMPUI_FLAGS_BORDER_RIGHT
+ AIMPUI_FLAGS_BORDER_BOTTOM

ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_GROUPBOX_PROPID_CAPTION
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_GROUPBOX_PROPID_TRANSPARENT
Свойство показывает, нужно ли рисовать фон у элемента.
0 - рисовать фон нужно.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_GROUPBOX_PROPID_CHECKMODE
Свойство позволяет включить отображение check box-а в заголовке группы.

  • AIMPUI_CHECKMODE_NONE - check box скрыт
  • AIMPUI_CHECKMODE_CUSTOM - check box показан
  • AIMPUI_CHECKMODE_TOGGLE_ENABLED - check box показан, изменение состояния меняет доступность (AIMPUI_CONTROL_PROPID_ENABLED) дочерних элементов
    • При изменении состояния срабатывает IAIMPUIChangeEvents
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_GROUPBOX_PROPID_CHECKED
    Состояние check box-а в заголовке группы (если AIMPUI_GROUPBOX_PROPID_CHECKMODE отлично от 0):
    Может принимать одно из следующих значений:
    • AIMPUI_CHECKSTATE_UNCHECKED
    • AIMPUI_CHECKSTATE_CHECKED
    • AIMPUI_CHECKSTATE_MIXED
    При изменении состояния срабатывает IAIMPUIChangeEvents
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    События
    IAIMPUIChangeEvents вызывается при изменении свойства AIMPUI_GROUPBOX_PROPID_CHECKED
    Создание экземпляра
    Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
    Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

IAIMPPlaylistGroup
Описание
Интерфейс IAIMPPlaylistGroup описывает группу плейлиста, в которой может находиться одна и более записей.
Предок
IAIMPPlaylistGroup наследуется от IAIMPPropertyList.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMP_PLAYLISTGROUP_PROPID_NAME
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPString, ..)
AIMP_PLAYLISTGROUP_PROPID_EXPANDED
Значение отличное от нуля - группа развернута
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_PLAYLISTGROUP_PROPID_DURATION
Суммарная длительность записей в группе, в секундах.
ТипDouble
ЧтениеIAIMPPropertyList.GetValueAsFloat
AIMP_PLAYLISTGROUP_PROPID_INDEX
Порядковый номер группы в плейлисте
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
AIMP_PLAYLISTGROUP_PROPID_SELECTED
Значение отличное от нуля - группа выбрана.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
Методы
GetItem
inIndexInteger Порядковый номер записи в группе
inIIDTGUID Идентификатор запрашиваемого интерфейса, должен быть равен IID_IAIMPPlaylistItem.
outObjectUntyped Экземпляр запрашиваемого интерфейса
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetItemCount
outResultInteger Количество записей в группе

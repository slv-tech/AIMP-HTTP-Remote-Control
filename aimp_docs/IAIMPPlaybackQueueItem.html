IAIMPPlaybackQueueItem
Описание
Экземпляр интерфейса содержит в себе расширенную информацию о позиции трека в очереди воспроизведения.
Предок
IAIMPPlaybackQueueItem наследуется от IAIMPPropertyList.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:
AIMP_PLAYBACKQUEUEITEM_PROPID_CUSTOM
Пользовательский параметр, позволяет прикрепить к экземпляру интерфейса пользовательский набор данных
ТипIUnknown
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IUnknown, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IUnknown>)
AIMP_PLAYBACKQUEUEITEM_PROPID_PLAYLISTITEM
Запись плейлиста, может быть равна nil / null, в этом случае воспроизведение остановится.
ТипIAIMPPlaylistItem
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPPlaylistItem, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPPlaylistItem>)
AIMP_PLAYBACKQUEUEITEM_PROPID_OFFSET
Стартовое смещение воспроизведения, в секундах. -1 - смещение не задано.
Позволяет запустить воспроизведение с определенной позиции.

Обратите внимание, свойство доступно только с v5.01.
ТипDouble
ЧтениеIAIMPPropertyList.GetValueAsFloat
ЗаписьIAIMPPropertyList.SetValueAsFloat

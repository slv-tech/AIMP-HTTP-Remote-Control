IAIMPUISplitter
Описание
Представляет собой разделитель (сплиттер) между элементами управления, путем перемещения которого можно изменить размер разделяемых элементов.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUISplitter
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_SPLITTER_PROPID_CANHIDE
Свойство указывает, можно ли скрывать управляемый элемент управления по двойному клику по сплиттеру.
0 - скрывать нельзя.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_SPLITTER_PROPID_CONTROL
Элемент управления, размером которого будет управлять сплиттер.
ТипIAIMPUIControl
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPUIControl, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPUIControl>)
События
IAIMPUIDrawEvents
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

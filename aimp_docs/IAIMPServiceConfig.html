IAIMPServiceConfig
Описание
IAIMPServiceConfig представляет собой интерфейс для работы с конфигурационным файлом программы.

Плагин должен использовать данный интерфейс для сохранения / загрузки своих собственных настроек.
Предок
IAIMPServiceConfig наследуется от IAIMPConfig
Методы
FlushCache
Метод позволяет сбросить все закэшированные данные на диск.
Метод медленный, не стоит вызывать его часто, это может отрицательно сказаться на производительности программы.
outResultHRESULT S_OK - Метод успешно завершился
E_ACCESSDENIED - Метод был вызван не из главного потока
E_UNEXPECTED - произошел непредвиденный сбой
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceConfig, ...);

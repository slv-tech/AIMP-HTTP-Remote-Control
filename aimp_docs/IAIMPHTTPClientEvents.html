IAIMPHTTPClientEvents
Описание
Интерфейс позволяет плагину получать уведомления от сделанного им активного запроса. Уведомления приходят от главного потока программы.
Методы
OnAccept
Событие возникает при получении HTTP-заголовка от сервера
inContentTypeIAIMPStringФормат содержимого ответа сервера
inContentSizeInt64 Размер содержимого, в байтах. Может быть равен 0, если размер содержимого неизвестен.
varAllowLongBool Плагин может отказаться принимать данные, для этого необходимо выставить данный параметр в FALSE
OnComplete
inErrorInfoIAIMPErrorInfo Если операция завершилась неудачно - ErrorInfo содержит дополнительную информацию об ошибке, в противном случае - ErrorInfo = nil/null
inCanceledLongBool True - операция была прервана по инициативе пользователя
OnProgress
inDownloadedInt64 Объем загруженных данных, в байтах
inTotalInt64 Общий объем данных, в байтах. Может быть равен 0, если объем неизвестен.
Создание экземпляра
Экземпляр создается плагином и передается в один из методов IAIMPServiceHTTPClient сервиса.

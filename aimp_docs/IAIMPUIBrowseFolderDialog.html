IAIMPUIBrowseFolderDialog
Описание
IAIMPUIBrowseFolderDialog предоставляет доступ диалогу выбора одной или нескольких папок
Методы
Execute
inOwnerWndHWND Дескриптор окна-владельца диалога.
Если указать 0 - будет использоваться дескриптор главного окна.
inFlagsLongWord 0 или комбинация из флагов:
+ AIMPUI_FLAGS_BROWSEFOLDER_CUSTOMPATHS - разрешить ввод пути вручную
+ AIMPUI_FLAGS_BROWSEFOLDER_MULTISELECT - разрешить выбор более одной папки
inDefaultPathIAIMPString Путь по умолчанию, диалог будет сфокусирован на нем при открытии. Может быть = nil / null
outSelectionIAIMPObjectList Список полных путей к выбранным папкам. Каждая запись листа представляет собой строку IAIMPString.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
Доступ к интерфейсу
<IAIMPServiceUI>.QueryInterface(IID_IAIMPUIBrowseFolderDialog, <IAIMPUIBrowseFolderDialog>)

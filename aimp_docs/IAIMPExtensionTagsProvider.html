IAIMPExtensionTagsProvider
Описание
Расширение для IAIMPServiceFindTagsOnline, позволяет плагину реализовать собственный провайдер тегов для функции "автозаполнения тегов с помощью онлайн-сервисов", что доступна в расширенном редакторе тегов.

Реализация интерфейса должна поддерживать асинхронную работу!

FindTracks
Получение списка треков по найденной информации об альбоме через метод FindAlbums.
Вы можете передать пользовательскую информацию через свойство IAIMPFileInfo.AIMP_FILEINFO_PROPID_CUSTOM.

Метод вызывается из второстепенного потока!

intOwnerIAIMPTaskOwner
intErrorInfoIAIMPErrorInfo Заполняется плагином в случае непредвиденной ошибке обработки данных.

E_UNEXPECTED - произошел непредвиденный сбой
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

IAIMPServiceLyrics
Описание
Сервис позволяет получить текст песни для указанного файла.
Поиск текста песни выполняется в отдельном потоке (опционально), плагин получит уведомление о завершении операции посредством функции обратного вызова.
Методы
Get
inFileInfoIAIMPFileInfo
inFlagsLongWord 0 или комбинация из флагов:
  • AIMP_SERVICE_LYRICS_FLAGS_NOCACHE - поиск обложки в обход кэша
  • AIMP_SERVICE_LYRICS_FLAGS_WAITFOR - метод не вернет управление до тех пор, пока операция не будет завершена
inCallbackProcTAIMPServiceLyricsReceiveProc Функция обратного вызова, через нее плагин получит результат.
inUserDataPointer Пользовательский параметр, передается в CallbackProc
outTaskTTaskHandle Идентификатор асинхронной задачи
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Cancel
Метод позволяет прервать активную задачу

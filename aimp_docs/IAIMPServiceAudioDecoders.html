IAIMPServiceAudioDecoders
Описание
Сервис предоставляет доступ к зарегистрированным в программе декодерам для аудиофайлов.
Методы
CreateDecoderForStream
Метод создает декодер для указанного потока
inStreamIAIMPStreamИнтерфейс потока
inFlagsLongWordДля будущего использования, должно быть равно нулю
inErrorInfoIAIMPErrorInfoИнформация об ошибке, заполняется плагином в случае ее возникновения.

Параметр может быть равен nil (null).
outDecoderIAIMPAudioDecoderЭкземпляр декодера
outResultHRESULT S_OK - Метод успешно завершился
E_FAIL - Декодер не смог открыть файл
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Декодер для файла не найден
E_UNEXPECTED - Произошел непредвиденный сбой
CreateDecoderForFileURI
Метод создает декодер для указанного виртуального файла
inFileURIIAIMPStringВиртуальное имя файла
inFlagsLongWordДля будущего использования, должно быть равно нулю
inErrorInfoIAIMPErrorInfoИнформация об ошибке, заполняется плагином в случае ее возникновения.

Параметр может быть равен nil (null).
outDecoderIAIMPAudioDecoderЭкземпляр декодера
outResultHRESULT S_OK - Метод успешно завершился
E_FAIL - Декодер не смог открыть файл
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Декодер для файла не найден
E_UNEXPECTED - Произошел непредвиденный сбой
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceAudioDecoders, ...);

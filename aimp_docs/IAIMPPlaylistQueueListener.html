IAIMPPlaylistQueueListener
Описание
Интерфейс позволяет плагину получать уведомления об изменениях в пользовательской очереди
Методы
ContentChanged
Событие возникает при изменении содержимого очереди
StateChanged
Событие возникает при изменении свойства AIMP_PLAYLISTQUEUE_PROPID_SUSPENDED у IAIMPPlaylistQueue.

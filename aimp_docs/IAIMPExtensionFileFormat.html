IAIMPExtensionFileFormat
Описание
IAIMPExtensionFileFormat представляет собой расширение для сервиса IAIMPServiceFileFormats, описывающее новый поддерживаемый программой формат файлов.
Методы
GetDescription
outSIAIMPString Краткое описание формата файла.
Например: Ogg Vorbis
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
GetExtList
outSIAIMPString Список поддерживаемых расширений.
Например: *.ogg;*.oga;

Обратите внимание, что ";" ставится всегда, даже у последнего в списке элемента.
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
GetFlags
outSIAIMPString Флаги, описывающие данный формат.
На текущий момент флаг может быть один из:
  • AIMP_SERVICE_FILEFORMATS_CATEGORY_AUDIO
    • ;
  • AIMP_SERVICE_FILEFORMATS_CATEGORY_PLAYLISTS
    • ;
    outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
    Создание экземпляра
    Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

IAIMPUIKeyboardEvents
Описание
IAIMPUIKeyboardEvents представляет собой интерфейс для обработки клавиатурных событий
Методы
OnEnter
Событие вызывается, когда элемент управления получает фокус.
inSenderIUnknown Элемент управления, вызвавший событие
OnExit
Событие вызывается, когда элемент управления теряет фокус.
inSenderIUnknown Элемент управления, вызвавший событие
OnKeyDown
inSenderIUnknown Элемент управления, вызвавший событие
varKeyWordВиртуальный код нажатой клавиши
inModifiersWord 0 или комбинация из следующих флагов:
  • AIMPUI_FLAGS_MOD_ALT
  • AIMPUI_FLAGS_MOD_CTRL
  • AIMPUI_FLAGS_MOD_SHIFT
OnKeyPress
inSenderIUnknown Элемент управления, вызвавший событие
varCharWideChar Введенный пользователем символ в кодировке UTF16
OnKeyUp
inSenderIUnknown Элемент управления, вызвавший событие
varKeyWordВиртуальный код нажатой клавиши
inModifiersWord 0 или комбинация из следующих флагов:
  • AIMPUI_FLAGS_MOD_ALT
  • AIMPUI_FLAGS_MOD_CTRL
  • AIMPUI_FLAGS_MOD_SHIFT
Создание экземпляра
Экземпляр создается плагином и передается в метод <IAIMPServiceUI>.CreateControl при создании элемента управления.

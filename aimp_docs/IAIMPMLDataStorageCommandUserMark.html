IAIMPMLDataStorageCommandUserMark
Описание
Команда представляет собой обработчик изменения данных пользователем в поле с зарезервированным именем AIMPML_RESERVED_FIELD_USERMARK.
Методы
SetMark
inIDOleVariant ID записи, для которой пользователь выставляет оценку
inValueDouble Оценка, от 0.0 .. 5.0
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.

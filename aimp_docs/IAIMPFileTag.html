IAIMPFileTag
Предок
IAIMPFileTag наследуется от IAIMPFileInfo.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMP_FILETAG_PROPID_TAG_ID
Тип тега, может принимать одно из следующих значений:
  • AIMP_FILETAG_ID_CUSTOM
  • AIMP_FILETAG_ID_APEv2
  • AIMP_FILETAG_ID_ID3v1
  • AIMP_FILETAG_ID_ID3v2
  • AIMP_FILETAG_ID_MP4
  • AIMP_FILETAG_ID_VORBIS
  • AIMP_FILETAG_ID_WMA
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_FILETAG_PROPID_DELETE_ON_SAVING
Свойство показывает, нужно ли удалять тег из файла при сохранении изменений. При значении, отличном от нуля, тег будет удален из файла.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32

IAIMPServicePlaylistManager
Описание
Сервис предоставляет доступ к плейлистам.
Методы
CreatePlaylist
inNameIAIMPString Имя плейлиста. Если не задано (равно nil/null) - плейлисту присваивается автоматическое имя.
inActivateLongBool Флаг показывает, нужно ли активировать только что созданный плейлист.
outPlaylistIAIMPPlaylist Интерфейс для доступа к созданному плейлисту
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

inFileNameIAIMPString Полное имя файла (см. Внешние и локальные плейлисты)

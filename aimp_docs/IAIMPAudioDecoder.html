IAIMPAudioDecoder
Методы
GetFileInfo
Метод получения метаданных для декодируемого файла.

Декодер не обязан заполнять метаданные, он может оставить это на откуп программе - программа получит информацию о файле встроенными средствами - для этого необходимо вернуть False в качестве результата.

Внимание! В реализации данного метода не допускается обращение к методам GetFileInfoXXX сервиса IAIMPServiceFileInfo.
inFileInfoIAIMPFileInfo
outResultLongBool True - если декодер заполнил FileInfo
GetStreamInfo
Метод возвращает информацию о звуковом потоке.
outSampleRateIntegerЧастота дискретизации
outChannelsIntegerКоличество каналов
outSampleFormatInteger Формат сэмплов, может принимать одно из значений:
  • AIMP_DECODER_SAMPLEFORMAT_08BIT
  • AIMP_DECODER_SAMPLEFORMAT_16BIT
  • AIMP_DECODER_SAMPLEFORMAT_24BIT
  • AIMP_DECODER_SAMPLEFORMAT_32BIT
  • AIMP_DECODER_SAMPLEFORMAT_32BITFLOAT
  • AIMP_DECODER_SAMPLEFORMAT_64BITFLOAT (доступно с v5.40)
outResultLongBool
IsSeekable
Флаг показывает, поддерживает ли декодер позиционирование по треку
outResultLongBool
IsRealTimeStream
Флаг показывает, идет исходный сигнал в реальном-времени или нет.
В случае реал тайм-сигнала становятся недоступными такие эффекты как: Speed, Tempo, Pitch.
outResultLongBool
GetAvailableData
Объем оставшихся данных для воспроизведения в байтах в формате декодированного потока.
В общем случае это значение должно быть равно GetSize - GetPosition.
outResultInt64
GetSize
Длительность в байтах в формате декодированного потока.
Может принимать нулевое значение в случае, если IsSeekable и/или IsRealTimeStream = True
outResultInt64
GetPosition
Текущая позиция в байтах в формате декодированного потока.
outResultInt64
SetPosition
inValueInt64 Новая позиция в байтах в формате декодированного потока.
outResultLongBool
Read
inBufferPByte Буфер для данных.
inCountInteger Указывает, сколько байт нужно считать
outResultInteger Количество считанных байт
Примечание 1
Реализация интерфейса должна поддерживать асинхронную работу.
Примечание 2
Обратите внимание, что все размеры указаны в байтах в формате уже декодированного (распакованного) потока.

Например:
Метод GetStreamInfo возвратил следующие значения для параметров: SampleRate = 44100, Channels = 2, SampleFormat = AIMP_DECODER_SAMPLEFORMAT_16BIT, тогда для 1-ой секунды GetPosition вернет: 1 * 44100 * 2 * (16 / 8) = 176400 байт.
Создание экземпляра
Экземпляр создается расширениями IAIMPExtensionAudioDecoder или IAIMPExtensionAudioDecoderOld вызове метода CreateDecoder.

TAIMPServiceAlbumArtReceiveProc
Описание
TAIMPServiceAlbumArtReceiveProc - функция обратного вызова для получения обложки альбома в результате асинхронного запроса. Функция вызывается главным потоком программы.
Методы
inImageIAIMPImage Экземпляр изображения. Может быть равен nil / null, в этом случае нужно обращаться к параметру ImageContainer.
inImageContainerIAIMPImageContainer Экземпляр контейнера изображения, может быть равен nil / null, если обложка альбома не найдена.
inUserDataPointer Пользовательский параметр, передается плагином вместе с функцией в вызываемый метод

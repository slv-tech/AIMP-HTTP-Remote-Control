IAIMPExtensionFileSystem
Описание
Расширение для сервиса IAIMPServiceFileSystems, должно поддерживать одну или несколько команд файловой системы (см. подраздел IAIMPServiceFileSystems).

Реализация интерфейса должна поддерживать асинхронную работу.
Предок
IAIMPExtensionFileSystem наследуется от IAIMPPropertyList.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMP_FILESYSTEM_PROPID_SCHEME
Схема путей файловой системы. 

схема://путь или схема:\\путь
Для большей информации см. IAIMPServiceFileSystems.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(.., IID_IAIMPString, ..)
AIMP_FILESYSTEM_PROPID_READONLY
Верните значение отличное от 0, если файловая система не поддерживает модификацию данных.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

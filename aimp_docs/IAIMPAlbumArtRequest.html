IAIMPAlbumArtRequest
Описание
IAIMPAlbumArtRequest расширение интерфейса IAIMPPropertyList, реализующее методы-помощники, упрощающие взаимодействие с настройками поиска, кэшем и загрузкой изображений.
Передается плагину от плеера через методы интерфейсов IAIMPMLAlbumArtProvider2 или IAIMPExtensionAlbumArtProvider3.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:
AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_FILES
0 - поиск обложки альбома среди файлов из папки с треком запрещен
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_FILES_MASKS
Список масок (без расширения) для поиска изображения среди файлов в папке.
Каждая маска отделяется от другого точкой с запятой.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_FILES_EXTS
Список поддерживаемых расширений.
Каждое расширение отделяется от другого точкой с запятой.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET
0 - поиск обложки альбома в интернете запрещен
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET_MAX_FILE_SIZE
Максимальный размер обложки альбома в байтах, которую IAIMPExtensionAlbumArtProvider может скачать из интернета.
0 - размер обложки неограничен.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_TAGS
0 - поиск обложки альбома в тегах файла запрещен.

Доступно с v5.02

Указывает число результатов поиска, которые может вернуть провайдер посредством вызова IAIMPAlbumArtRequest.CachePut в ходе одного запроса.
Только для провайдеров категории AIMP_ALBUMART_PROVIDER_CATEGORY_INTERNET_SEARCH.

Методы
CacheGet
Возвращает IAIMPImageContainer для указанного ключа.
Обратите внимание, что несмотря на успешное выполнение функции, Image может быть равен nil / null.
inKeyIAIMPStringСтрока-ключ должна быть построена так, чтобы однозначно идентифицировать обложку альбома для указанных IAIMPFileInfo.
varImageIAIMPImageContainer
outResultHRESULT S_OK - операция успешно завершилась
E_FAIL - записи с таким ключом нет в кэше
E_ACCESSDENIED - кэш отключен настройками программы или параметрами поиска
E_UNEXPECTED - произошел непредвиденный сбой
CachePut
Оптимизирует переданную в виде IAIMPImageContainer картинку и записывает его в кэш.
В качестве Image может быть передан nil / null, если нужно закэшировать "отсутствующее значение".

inKeyIAIMPStringСтрока-ключ должна быть построена так, чтобы однозначно идентифицировать обложку альбома для указанных IAIMPFileInfo.
varImageIAIMPImageContainer Функция вернет оптимизированную копию картинки, если флаг AIMP_SERVICE_ALBUMART_FLAGS_ORIGINAL не был указан при создании запроса.
outResultHRESULT S_OK - операция успешно завершилась
E_UNEXPECTED - произошел непредвиденный сбой
Download
Функция-помощник по загрузке изображений по указанному URL.
Самостоятельно учитывает настроек поиска: AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET и AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET_MAX_FILE_SIZE.
inURLIAIMPString Поддерживаемые протоколы: file:// и http(s)://
outImageIAIMPImageContainer
outResultHRESULT S_OK - операция успешно завершилась
E_FAIL - операция завершилась с ошибкой
E_ABORT - операция отменена пользователем
E_ACCESSDENIED - загрузка обложек с интернета отключена настройками программы или параметрами поиска
E_OUTOFMEMORY - размер картинки превышает лимит, установленный опцией AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET_MAX_FILE_SIZE
E_UNEXPECTED - произошел непредвиденный сбой
IsCanceled
Выполняет сервисную функцию, позволяет плагину узнать об отмене выполняемой задачи
outResultLongBool Если True - плагин должен прервать выполнение задачи.
Пример использования
Рассмотрим взаимодействие с IAIMPAlbumArtRequest на примере провайдера, скачивающего картинку из интернета:

function Get(FileInfo: IAIMPFileInfo; Request: IAIMPAlbumArtRequest; out Image: IAIMPImageContainer): HRESULT; stdcall;
var
  ALink: IAIMPString;
begin
  Result := E_FAIL;
  // Проверяем, задана ли у файла ссылка?
  if PropListGetStr(FileInfo, AIMP_FILEINFO_PROPID_URL, ALink) then
  begin 
    // Это ссылка на картинку?
    if IsImageURL(ALink) then 
    begin
      // Она уже есть в кэше? 
      if Succeeded(Request.CacheGet(ALink, Image)) then
        Exit(S_OK);
        
      // Скачиваем картинку
      // Функция Download самостоятельно проверит и учтёт значения опций 
      // AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET и AIMP_ALBUMART_REQUEST_PROPID_FIND_IN_INTERNET_MAX_FILE_SIZE
      if Succeeded(Request.Download(ALink, Image)) then
      begin
        // Оптимизируем и обновляем кэш 
        Request.CachePut(ALink, Image);
        Result := S_OK;
      end;
    end;
  end;
end;

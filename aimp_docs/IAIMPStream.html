IAIMPStream
Описание
IAIMPStream представляет интерфейс для работы с потоками данных.
Методы
GetSize
outResultInt64Объем хранимых в потоке данных, в байтах
SetSize
Метод позволяет заранее указать вместимость потока, в байтах.

Примечание: размер потока будет автоматически увеличиваться при записи в него данных в случае необходимости. Данный метод служит лишь для оптимизации - он позволяет уменьшить число переаллокаций внутренних ресурсов потока при изменении вместимости.

Внимание! Если текущая вместимость потока больше той, что устанавливается методом - часть данных будет выброшена.
inValueInt64Новая вместимость потока, в байтах
outResultHRESULT S_OK - Вызов завершился без ошибок
E_NOTIMPL - Поток не поддерживает запись
E_UNEXPECTED - Произошла непредвиденная ошибка
GetPosition
outResultInt64Позиция курсора чтения/записи, в байтах
Seek
Метод позволяет изменить положение курсора чтения/записи.
inOffsetInt64Смещение курсора, в байтах
inModeInteger Параметр указывает, как трактовать смещение, заданное в параметре Offset.
Может принимать одно из следующих значений:
  • AIMP_STREAM_SEEKMODE_FROM_BEGINNING - смещение указано относительно начала потока
  • AIMP_STREAM_SEEKMODE_FROM_CURRENT - смещение указано относительно текущей позиции курсора
  • AIMP_STREAM_SEEKMODE_FROM_END - смещение указано относительно конца потока
outResultHRESULT S_OK - Вызов завершился без ошибок
E_FAIL - Смещение по указанному Offset-у не было выполнено. Например, такое возможно при выходе курсора за границы потока.
E_INVALIDARG - Аргумент Mode был задан неверно
E_UNEXPECTED - Произошла непредвиденная ошибка
Read
varBufferPByte Буфер, в который будут записаны данные. Указатель на массив байт.
inCountLongWord Сколько байт нужно записать в буфер
outResultInteger-1 в случае непредвиденной ошибки, иначе - количество байт, что были записаны в буфер.
Write
inBufferPByte Буфер, данные из которого будут записаны в поток. Указатель на массив байт.
inCountLongWord Сколько байт нужно записать в поток
outWrittenPInteger Указатель на Integer.
Показывает, сколько байт было записано в поток.
Параметр необязательный - может быть равен nil / NULL.
outResultHRESULT S_OK - Вызов завершился без ошибок
E_NOTIMPL - Поток не поддерживает запись
E_UNEXPECTED - Произошла непредвиденная ошибка

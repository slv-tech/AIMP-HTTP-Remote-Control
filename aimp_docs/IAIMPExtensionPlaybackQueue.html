IAIMPExtensionPlaybackQueue
Описание
Это расширение сервиса IAIMPServicePlaybackQueue позволяет плагину переопределить внутреннюю очередь воспроизведения треков.

Внимание! Следующий / предыдущий трек должен вычисляться строго по значению параметра Current. В общем случае, Current может не совпадать с текущим проигрываемым треком. Так же, плеер может несколько раз запрашивать следующий файл для одного и того же значения параметра Current, очень важно, чтобы плагин во всех случаях выдавал одно и тоже значение.

Плагин не должен учитывать значение свойства AIMP_PLAYLISTITEM_PROPID_PLAYINGSWITCH IAIMPPlaylistItem-а при поиске следующего / предыдущего трека, т.к. это делается на стороне плеера.
Методы
GetNext
Событие вызывается при запросе следующего для воспроизведения трека
inCurrentIUnknown Ссылка на текущий элемент очереди, от которого вычисляется следующий (см. Flags).
В общем случае, Current может не совпадать с текущим проигрываемым треком.
inFlagsLongWord Флаги показывают, как интерпретировать значение Current.
На текущий момент переменная может содержать один из флагов:
  • AIMP_PLAYBACKQUEUE_FLAGS_START_FROM_BEGINNING - очередь должна вернуть самую первую запись. Current - IAIMPPlaylist
  • AIMP_PLAYBACKQUEUE_FLAGS_START_FROM_CURSOR - очередь должна вернуть следующую за курсором воспроизведения запись. Current - IAIMPPlaylist
  • AIMP_PLAYBACKQUEUE_FLAGS_START_FROM_ITEM - очередь должна вернуть следующую за указанной в Current запись. Current - IAIMPPlaylistItem
inQueueItemIAIMPPlaybackQueueItem Экземпляр интерфейса, содержащий в себе информацию о треке, заполняется плагином.
outResultLongBool False - очередь отключена.

Обратите внимание, что даже в том случае, если очередь не смогла найти запись, удовлетворяющую условиям поиска, метод должен заполнить QueueItem и вернуть True в качестве результата.

OnSelect
Событие вызывается при попытке проиграть выбранный Item. Плагин не должен менять свойства Item-а.
inItemIAIMPPlaylistItem
inQueueItemIAIMPPlaybackQueueItem
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

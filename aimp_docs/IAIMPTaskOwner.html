IAIMPTaskOwner
Описание
Интерфейс выполняет сервисную функцию, позволяет плагину узнать об отмене выполняемой задачи. Экземпляр интерфейса создается программой и передается плагину при вызове IAIMPTask.Execute
Методы
IsCanceled
outResultLongBool Если True - плагин должен прервать выполнение задачи.

Резюме

Объявление:apiThreading.pas / apiThreading.h
Асинхронная работа:Поддерживается
Приложения:Player, Audio Converter, Advanced Tag Editor

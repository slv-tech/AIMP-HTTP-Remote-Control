IAIMPUITreeList
Описание
IAIMPUITreeList представляет собой таблицу с поддержкой древовидной структуры данных.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUITreeList
Свойства
AIMPUI_TL_PROPID_ALLOW_DELETING
Разрешает пользователю удаление записей из списка.
0 - запрещено.

IAIMPServiceConnectionSettings
Описание
Сервис предоставляет доступ к текущим настройкам подключения.
Предок
IAIMPServiceConnectionSettings наследуется от IAIMPPropertyList.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMP_SERVICE_CONSET_PROPID_CONNECTION_TYPE
Тип подключения, может принимать одно из значений:
  • AIMP_SERVICE_CONSET_CONNECTIONTYPE_DIRECT - прямое подключение к интернет
  • AIMP_SERVICE_CONSET_CONNECTIONTYPE_PROXY - подключение к интернет через прокси сервер, заданный в настройках программы
  • AIMP_SERVICE_CONSET_CONNECTIONTYPE_SYSTEMDEFAULTS - подключение к интернет, используя текущие системные настройки
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_SERVICE_CONSET_PROPID_PROXY_SERVER
Символьный или IP адрес прокси-сервера, только для ConnectionType = AIMP_SERVICE_CONSET_CONNECTIONTYPE_PROXY
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_SERVICE_CONSET_PROPID_PROXY_PORT
Порт прокси сервера, только для ConnectionType = AIMP_SERVICE_CONSET_CONNECTIONTYPE_PROXY
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_SERVICE_CONSET_PROPID_PROXY_USERNAME
Имя пользователя. Может отсутствовать, только для ConnectionType = AIMP_SERVICE_CONSET_CONNECTIONTYPE_PROXY
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_SERVICE_CONSET_PROPID_PROXY_USERPASS
Пароль. Может отсутствовать, только для ConnectionType = AIMP_SERVICE_CONSET_CONNECTIONTYPE_PROXY
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_SERVICE_CONSET_PROPID_TIMEOUT
Максимальное время ожидания, в миллисекундах.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_SERVICE_CONSET_PROPID_USERAGENT
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceConnectionSettings, ...);

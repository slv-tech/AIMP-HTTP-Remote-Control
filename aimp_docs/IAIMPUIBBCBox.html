IAIMPUIBBCBox
Описание
IAIMPUIBBCBox позволяет выводить текст с форматированием с помощью BB-кодов.
Поддерживаются следующие BB-коды: B, I, U, S, COLOR, URL.
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIBBCBox
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_BBCBOX_PROPID_AUTOSCROLL
Определяет поведение в ситуации, когда текст не влезает полностью в контрол:
0 - текст будет обрезан с многоточием в конце
1 - появятся ползунки для прокрутки текста (по умолчанию).

Доступно с v5.40
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BBCBOX_PROPID_BORDERS
Свойство указывает, у каких сторон нужно рисовать рамку.

0 или комбинация из флагов:
+ AIMPUI_FLAGS_BORDER_LEFT
+ AIMPUI_FLAGS_BORDER_TOP
+ AIMPUI_FLAGS_BORDER_RIGHT
+ AIMPUI_FLAGS_BORDER_BOTTOM

ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_BBCBOX_PROPID_TEXT
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_BBCBOX_PROPID_TEXTALIGN
Выравнивание текста по горизонтали, может принимать одно из значений:
  • AIMPUI_ALIGN_NEAR - (left) (по умолчанию)
  • AIMPUI_ALIGN_CENTER
  • AIMPUI_ALIGN_FAR - (right)

    • Доступно с v5.40
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_BBCBOX_PROPID_TEXTCOLOR
    Цвет текста в формате RRGGBB

    Доступно с v5.40
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_BBCBOX_PROPID_TRANSPARENT
    Свойство показывает, нужно ли рисовать фон у элемента.
    0 - рисовать фон нужно.
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_BBCBOX_PROPID_PADDING
    Отступ от краев элемента управления до его содержимого. От 0 до 100.
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    AIMPUI_BBCBOX_PROPID_WORDWRAP
    Свойство показывает, можно ли переносить текст по словам, если он не влезает.
    0 - перенос по словам выключен.
    ТипInteger
    ЧтениеIAIMPPropertyList.GetValueAsInt32
    ЗаписьIAIMPPropertyList.SetValueAsInt32
    Создание экземпляра
    Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
    Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

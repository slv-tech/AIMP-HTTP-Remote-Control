IAIMPServiceMessageDispatcher
Описание
Общение между модулями программы осуществляется посредством сообщений. Данный сервис предоставляет доступ к механизму обмена сообщениями.
Методы
Send
inMessageLongWord Идентификатор сообщения.
См. разделы "Свойства", "События", "Команды".
inParam1Integer
inParam2Pointer
outResultHRESULT
Register
Позволяет зарегистрировать собственное сообщение
inMessageNamePChar Имя сообщения
outResultLongWord Идентификатор сообщения, зависит от имени. Для одинаковых имен функция вернет одинаковые значения
Hook
Установка перехватчика
inHookIAIMPMessageHook
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Unhook
Удаление перехватчика
inHookIAIMPMessageHook
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
Доступ к сервису
<IAIMPCore>.QueryInterface(IID_IAIMPServiceMessageDispatcher, ...);

IAIMPUIMemo
Описание
IAIMPUIMemo представляет собой многострочный редактор текста
Предок
IAIMPPropertyList -> IAIMPUIControl -> IAIMPUIWinControl -> IAIMPUIBaseEdit -> IAIMPUIMemo
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_MEMO_PROPID_CARET_XY
Позиция каретки в редакторе: младшее слово Integer - Х, старшее слово Integer - Y.
ТипInt64
ЧтениеIAIMPPropertyList.GetValueAsInt64
ЗаписьIAIMPPropertyList.SetValueAsInt64

События
IAIMPUIChangeEvents вызывается при изменении текста
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

IAIMPMLGroupingTreeDataProvider2
Описание
IAIMPMLGroupingTreeDataProvider2 необязательное расширение провайдера IAIMPMLGroupingTreeDataProvider.
Методы
GetPathSeparator
outResultChar Позволяет указать разделитель пути в дереве группировки.

Если плагин не реализует этот интерфейс - плеер будет использовать разделитель, характерный для текущей ОС.

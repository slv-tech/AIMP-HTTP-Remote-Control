TAIMPUIControlPlacementConstraints
Описание
TAIMPUIControlPlacementConstraints позволяет указать минимальные / максимальные размеры для элементов управления. Укажите 0 в качестве значения поля, чтобы снять ограничение.
Создание экземпляра
Экземпляр создается с помощью <IAIMPServiceUI>.CreateControl.
Элемент управления удаляется автоматически при убиении окна, на котором оно расположено (см. IAIMPUIForm).

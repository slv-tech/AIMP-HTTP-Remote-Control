Схема взаимодействия
Взаимодействие плеера и стороннего приложения происходит посредством обмена сообщениями с помощью родного механизма ОС Windows.

При запуске плеера создается специальное невидимое окно с классом AIMPRemoteAccessClass, которое служит для обмена сообщениями со сторонними приложениями.
Описание констант
Описание всех констант расположено в apiRemote.pas / apiRemote.h

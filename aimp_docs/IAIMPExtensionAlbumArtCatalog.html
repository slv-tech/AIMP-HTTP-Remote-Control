IAIMPExtensionAlbumArtCatalog
Описание
IAIMPExtensionAlbumArtCatalog представляет собой расширение для сервиса IAIMPServiceAlbumArt, и позволяет плагину добавить пункт в контекстное меню внешних каталогов обложек альбомов. Конечный пользователь увидит это меню, кликнув ПКМ на дисплее обложки альбома в редакторе тегов.
Методы
GetIcon
Метод возвращает иконку каталога размером 16х16 для пункта меню
outImageHICON Дескриптор иконки в терминах Win32 API
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
GetName
Метод возвращает имя каталога для пункта меню
outNameIAIMPString
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
Show
Метод вызывается при клике на пункте меню
inFileURIIAIMPStringFileURI
inArtistIAIMPString Исполнитель альбома. Опционально, может быть равен nil / null
inAlbumIAIMPString Название альбома. Опционально, может быть равен nil / null
outImageIAIMPImageContainer Обложка альбома, может быть равна nil / null, если каталог не предусматривает непосредственного выбора. Например, расширение перенаправляет вызов на какой-либо сетевой ресурс.
outResultHRESULTS_OK - в случае успеха, в случае ошибки - любое другое значение.
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.

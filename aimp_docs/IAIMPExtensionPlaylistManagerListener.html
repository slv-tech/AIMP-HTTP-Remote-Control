IAIMPExtensionPlaylistManagerListener
Описание
Расширение для IAIMPServicePlaylistManager, позволяет плагину получать уведомления об активации, добавлении и удалении плейлистов.
Методы
PlaylistActivated
Событие возникает, когда какой-либо плейлист активируется
inPlaylistIAIMPPlaylist
PlaylistAdded
Событие возникает после создания / загрузки плейлиста
inPlaylistIAIMPPlaylist
PlaylistRemoved
Событие возникает перед удалением / выгрузкой плейлиста
inPlaylistIAIMPPlaylist
Создание экземпляра
Экземпляр создается плагином и регистрируется в программе при инициализации плагина при помощи IAIMPCore.RegisterExtension с указанием соответствующего идентификатора сервиса.
Резюме

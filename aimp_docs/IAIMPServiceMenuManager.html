IAIMPServiceMenuManager
Описание
Сервис предоставляет доступ к уже созданным пунктам меню.
Методы
GetBuiltIn
Метод возвращает интерфейс для встроенного пункта меню.
inIDInteger Доступны следующие идентификаторы:

Желтые маркеры означают, что новый пункт будет добавляться в указанное место. Красные - в конец подменю.

















AIMP_MENUID_PLAYER_EQ_LIB - контекстное меню с настройками эквалайзера.

AIMP_MENUID_PLAYER_STOP_OPTIONS - контекстное меню кнопки "остановить воспроизведение". Внимание: меню введено в v4.00.1691
outMenuItemIAIMPMenuItem Экземпляр интерфейса доступа к пункту меню. Только для чтения.
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока
GetByID
inIDIAIMPString Уникальный идентификатор пункта меню. См. AIMP_MENUITEM_PROPID_ID
outMenuItemIAIMPMenuItem Экземпляр интерфейса доступа к пункту меню
outResultHRESULTS_OK - Метод успешно завершился
E_INVALIDARG - Один из параметров задан неверно
E_NOTIMPL - Метод не реализован
E_UNEXPECTED - Произошел непредвиденный сбой
E_ACCESSDENIED - Метод был вызван не из главного потока

IAIMPAction
Описание
IAIMPAction описывает команду общего назначения. Основной особенностью команды является возможность отображения в списке горячих клавиш, и, как следствие, возможность изменения пользователем локальных и глобальных горячих клавиш для этой команды.
Предок
IAIMPAction наследуется от IAIMPPropertyList.
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMP_ACTION_PROPID_CUSTOM
Пользовательский параметр, позволяет прикрепить к экземпляру интерфейса пользовательский набор данных
ТипIUnknown
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IUnknown, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IUnknown>)
AIMP_ACTION_PROPID_ID
Уникальный идентификатор команды, должен однозначно идентифицировать команду среди всех плагинов
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_ACTION_PROPID_NAME
Имя команды, отображается пользователю в диалоге настройки горячих клавиш.

См. автоматическая локализация
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_ACTION_PROPID_GROUPNAME
Имя группы, в которую входит команда, необходимо для отображения в диалоге настроек горячих клавиш.

См. автоматическая локализация
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMP_ACTION_PROPID_ENABLED
Состояние команды: 0 - недоступна, иначе - любое другое значение
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_ACTION_PROPID_DEFAULTLOCALHOTKEY
Локальная горячая клавиша по умолчанию (может быть равно 0), см. IAIMPServiceActionManager.MakeHotkey
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_ACTION_PROPID_DEFAULTGLOBALHOTKEY
Глобальная горячая клавиша по умолчанию (может быть равно 0), см. IAIMPServiceActionManager.MakeHotkey
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_ACTION_PROPID_DEFAULTGLOBALHOTKEY2
Второстепенная глобальная горячая клавиша по умолчанию (может быть равно 0), см. IAIMPServiceActionManager.MakeHotkey
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMP_ACTION_PROPID_EVENT
Обработчик вызова команды.
В качестве параметра Data события IAIMPActionEvent передается интерфейс вызывающей команды в виде IAIMPAction
ТипIAIMPActionEvent
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPActionEvent, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPActionEvent>)
Отображение в настройках горячих клавиш
Только команды с заполненными AIMP_ACTION_PROPID_ID, AIMP_ACTION_PROPID_NAME и AIMP_ACTION_PROPID_GROUPNAME свойствами отображаются в настройках горячих клавиш.
Создание экземпляра
Экземпляр создается с помощью <IAIMPCore>.CreateObject(IID_IAIMPAction, ...) при инициализации плагина. После заполнения необходимых полей экземпляра, плагин обязан зарегистрировать команду в менеджере с помощью <IAIMPCore>.RegisterExtension(IID_IAIMPServiceActionManager, <IAIMPAction>). Команда удаляется автоматически при финализации плагина.

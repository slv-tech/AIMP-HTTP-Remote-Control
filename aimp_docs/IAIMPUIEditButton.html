IAIMPUIEditButton
Описание
IAIMPUIEditButton описывает кнопку, что расположена внутри однострочного поля ввода
Предок
IAIMPPropertyList -> IAIMPUIEditButton
Свойства
Доступные идентификаторы свойств для методов интерфейса IAIMPPropertyList:

AIMPUI_EDITBUTTON_PROPID_CUSTOM
Пользовательский параметр, позволяет прикрепить к экземпляру интерфейса пользовательский набор данных
ТипIUnknown
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IUnknown, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IUnknown>)
AIMPUI_EDITBUTTON_PROPID_CAPTION
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_EDITBUTTON_PROPID_ENABLED
Доступность элемента: 0 - недоступен, иначе - любое другое значение.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_EDITBUTTON_PROPID_HINT
Текст подсказки, отображается при удержании мыши над элементом.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
AIMPUI_EDITBUTTON_PROPID_INDEX
Порядковый номер кнопки в списке
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_EDITBUTTON_PROPID_IMAGEINDEX
Индекс картинки в IAIMPUIImageList для отрисовки.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_EDITBUTTON_PROPID_VISIBLE
Видимость элемента: 0 - невиден, иначе - любое другое значение.
ТипInteger
ЧтениеIAIMPPropertyList.GetValueAsInt32
ЗаписьIAIMPPropertyList.SetValueAsInt32
AIMPUI_EDITBUTTON_PROPID_WIDTH
Размер кнопки, от 1 пкс.
ТипIAIMPString
ЧтениеIAIMPPropertyList.GetValueAsObject(ID, IID_IAIMPString, ..)
ЗаписьIAIMPPropertyList.SetValueAsObject(ID, <IAIMPString>)
События
IAIMPUIChangeEvents вызывается при клике по кнопке
